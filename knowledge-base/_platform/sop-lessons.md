### [2026-03-21 / 헤이두두 2차 재테스트 — 검증 완료]

**검증 결과: 1차에서 발견된 14개 이슈 100% 해결 확인.**

신규 발견 (향후 개선 대상):
1. **캠페인 기획 SOP**: 시즌 미지정 시 자동으로 S/S 가정 — 명시적 fallback 필요.
2. **유아동복 퍼널 전환율 벤치마크**: 공개 데이터 부재 — 업종별 벤치마크 DB 구축 필요.
3. **상세페이지 Q&A 6개 반복**: 기능 6개가 동일 구조(Q→A→이미지→혜택)라 단조로움 위험 — 2-3개 변형 패턴 추가 검토.
4. **카드뉴스 채널**: 소셜 캘린더에서 카드뉴스를 공식 채널로 포함해야 함.
5. **B2C 아웃바운드**: 콜드 세일즈가 아닌 "채널 입점 제안"으로 재정의 필요.

### [2026-03-21 / 모윰 테스트 — 2번째 클라이언트]

**학습 루프 검증: Platform KB 교훈 10/10 자동 적용 확인. 재테스트 불필요.**

신규 발견:
1. **멀티채널 가격 일관성**: 쿠팡/SSG/G마켓/롯데ON 등 5개+ 마켓플레이스에서 판매 시, 채널별 가격 정책 관리 규칙 필요. pricing-rules.md에 "채널별 가격 정책" 섹션 추가 검토.
2. **상시 할인 리스크**: 30-50% 상시 할인은 브랜드 가치 훼손 위험. pricing-strategy SOP에 "상시 할인 리스크 체크리스트" 추가 검토.
3. **서브 브랜드 아키텍처**: 모윰/모브아(MOV.aa) 같은 서브 브랜드가 있을 때 포지셔닝/메시지 분리 규칙 필요.
4. **글로벌 시장 SOP 부재**: 해외 진출(아마존 US, 동남아, 중동) 시 hreflang/다국어 SEO 전략 SOP 필요.
5. **수상/인증 자산 활용**: 디자인 어워드, 소아치과학회 인증 등을 E-E-A-T 신호로 체계적 매핑하는 프레임워크 필요.
6. **기능성 제품 팩트체크**: 소재(Duradex ECHO PPSU) 등 기술적 주장의 팩트체크 프로세스 필요.
7. **의류 vs 용품 카피 차이**: 의류(감성/디자인 중심) vs 용품(안전/기능/인증 중심) — 같은 유아 B2C라도 카피 톤이 다름.

### [2026-03-21 / 업타운 빈티지 테스트 — 3번째 클라이언트, 플랫폼 한계 발견]

**테스트 결과: 플랫폼 호환성 심각한 한계 발견. 3가지 비즈니스 유형에 대한 근본적 SOP 확장 필요.**

1. **마켓플레이스 네이티브 브랜드 (호환 4/10)**: 자사몰 없이 네이버 스마트스토어만 운영하는 브랜드에서 GA4 감사, Google SEO, 랜딩페이지 SOP가 전부 부적합. 네이버 쇼핑 SEO(키워드/리뷰/찜/판매량 알고리즘)가 별도 SOP로 필요.
2. **일품 재고 비즈니스 (호환 3/10)**: 빈티지/구제는 모든 상품이 1점뿐. 가격 전략(표준 가격표 불가), 캠페인(재입고/리마케팅 불가), 퍼널(상품 페이지가 수시로 소멸) SOP가 근본적으로 부적합.
3. **마이크로 비즈니스 (호환 3/10)**: 1-3인 운영에게 11개 에이전트 풀 산출물은 과잉. "Lite 모드"(핵심 3-5개 SOP만 실행) 필요.
4. **네이버 스마트스토어 WebFetch 실패**: JS 렌더링 기반이라 WebFetch 불가. 네이버 쇼핑 API 또는 간접 수집 fallback 필요.
5. **비주얼 콘텐츠 SOP 부재**: 빈티지 패션에서 상품 촬영이 곧 마케팅. 촬영 가이드/스타일링 가이드 SOP 필요.
6. **C2C 경쟁사 분석 프레임워크 부재**: 번개장터/당근마켓 같은 C2C 플랫폼이 주요 경쟁자이나 기존 competitor SOP로 분석 불가.

### [2026-03-21 / 업타운 빈티지 재테스트 — 빈티지 가구점 (수정)]

**호환성 3/10 → 7/10 향상 확인. 신규 3개 플레이북(일품 재고/네이버 쇼핑 SEO/비주얼 가이드) 효과 검증.**

1. **일품 재고 전략 SOP 효과**: FOMO 마케팅, 큐레이션 브랜딩, 카테고리 SEO, 실시간 콘텐츠 워크플로우 — 빈티지 가구에 높은 적합도.
2. **네이버 쇼핑 SEO SOP 효과**: 상품명 공식, 리뷰 확보 전략, CTR 최적화 — 스마트스토어 셀러에게 즉시 실행 가능.
3. **비주얼 콘텐츠 가이드 효과**: 가구 촬영 6샷 시스템(전체/디테일/소재/스케일/스타일링/컨디션), 파티나 촬영법 — 패션과 완전히 다른 촬영 규칙 필요 확인.
4. **상세페이지 24섹션 → 18섹션 축소**: 가구는 6개 기능 Q&A 대신 "6개 스토리 씬"(소재/디자인/기능/컨디션/공간/관리)으로 변형 필요.
5. **오프라인+온라인 O2O 퍼널**: 가구는 고가+실물 확인 필요 → 온라인 발견 → 매장 방문 예약 → 현장 구매 퍼널이 핵심. 기존 퍼널 SOP에 O2O 변형 추가 검토.
6. **업종별 카피 차이 재확인**: 가구(소재/디자이너/연대/컨디션 중심) ≠ 의류(감성/디자인 중심) ≠ 용품(안전/인증 중심)
