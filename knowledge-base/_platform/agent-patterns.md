# 에이전트 실행 패턴 (Platform-Level)

> 에이전트 실행 시 효과적이었던 패턴과 실패 패턴을 축적합니다.
> 에이전트는 작업 전 이 파일을 참조하여 검증된 패턴을 우선 적용합니다.
> Append-only. 항목당 최대 300자.

### [2026-03-21 / 검증된 성공 패턴]

1. **JSON 핸드오프 > 마크다운 전문 전달**: 에이전트 간 데이터 전달 시 JSON 핸드오프가 토큰 비용 절감 + 필드 정확도 향상. summary(3줄) + key_findings + data 구조가 효과적.
2. **자체 검증 체크리스트**: reviewer 전 자체 체크로 FAIL 재실행 비용 50% 이상 절감 기대.
3. **소스 신뢰도 3등급**: high/medium/low 태깅이 출처 품질 추적에 효과적. 소규모 브랜드 특례가 현실적으로 필수.
4. **B2C 플랫폼 선택 로직**: config.md의 B2B/B2C 비중으로 자동 분기하는 패턴이 모든 콘텐츠 SOP에 적용 가능.
5. **Strategy SOP 체인 실행**: GTM→포지셔닝→채널→퍼널→가격→캠페인 순서가 산출물 품질을 크게 높임. 이전 SOP 결과가 다음 입력이 됨.

### [2026-03-21 / 검증된 실패 패턴 (회피)]

1. **B2B 가정 기본값**: SOP/에이전트의 기본 가정이 B2B SaaS면 B2C 클라이언트에서 50%+ 항목이 부적합. 반드시 B2B/B2C 분기 필요.
2. **필수 입력 미확인 시 중단**: 예산/KPI가 없다고 SOP를 중단하면 아무것도 생산 못함. 비중(%)로 대체 + 워터마크가 더 실용적.
3. **3채널 고정 출력**: LinkedIn/Social Channel A/Twitter 고정은 B2C에서 비효율. 활성 채널 기반 동적 선택 필수.
4. **온보딩 시 config.md만 생성**: brand-guidelines, copy-patterns, pricing-rules 미생성 시 모든 후속 SOP에 연쇄 영향.

### [2026-03-21 / 2차 재테스트 — 신규 검증 패턴]

1. **B2C 플랫폼 분기 확인됨**: config.md의 B2B/B2C 비중 → B2C>50%이면 Social Channel A+네이버블로그 기본, LinkedIn 제외. 30개 산출물에서 일관 작동.
2. **예산 fallback 패턴 확인됨**: [미확인] 워터마크 + 비중(%)만 제시 패턴이 전략 6개 SOP 전체에서 정상 작동.
3. **200자 이메일 제한 확인됨**: 아웃바운드 3통 모두 200자 이하 (162/170/148자). 간결성이 오히려 품질 향상.
4. **GEO 스코어링 통합 확인됨**: SEO 감사에 GEO 8/30 포함. AI 인용 가능성, E-E-A-T, 크롤러 접근이 별도 카테고리로 분리되어 실행됨.
5. **고객 스토리 변형 확인됨**: B2C에서 SCSR 대신 5단계(고객소개→구매전고민→선택이유→사용경험→추천) 구조가 자연스럽게 작동.

### [2026-03-21 / Sample Client D 테스트 — 학습 루프 2회차]

**자기 개선 루프 성과: Sample Client C 교훈 → Sample Client D에 100% 자동 적용 → 재테스트 불필요**

1. **Platform KB 학습 루프 작동 확인**: 2번째 클라이언트에서 동일 실수 0건. 교훈 축적→참조→적용 사이클이 정상 작동.
2. **유아용품 B2C 패턴**: 젖병/빨대컵 등 기능성 제품은 안전/인증/소재가 카피 핵심. 의류(감성/디자인)와 다름. 업종 세분화 필요.
3. **멀티 마켓플레이스 브랜드**: 5개+ 채널 판매 시 가격 일관성 관리가 핵심 과제. 단일 채널 D2C와 전략이 다름.
4. **글로벌 진출 브랜드**: 해외 시장이 있으면 SEO에 hreflang, 다국어 콘텐츠, 현지화 전략이 추가로 필요.
5. **포지셔닝 축 패턴**: 유아 B2C 카테고리(의류+용품 모두)에서 "디자인 감성 x 소재 품질" 축이 범용적으로 작동. 업종별 범용 축 DB 구축 가능성.
6. **아웃바운드 200자 제한 재확인**: Sample Client D 183/173/145자 — Sample Client C 162/170/148자와 유사 범위. 200자 제한이 현실적으로 작동.

### [2026-03-21 / Sample Client E 테스트 — 플랫폼 한계 패턴]

1. **플랫폼 호환성 매트릭스 발견**: 현재 SOP는 "자사몰 보유 + 대량생산 + 중소기업" 기준 설계. 이 3조건 중 하나라도 벗어나면 호환성 급락.
2. **비즈니스 유형별 호환 점수**: D2C 자사몰(9/10), 마켓플레이스 네이티브(4/10), 일품 재고(3/10), 마이크로 1-3인(3/10)
3. **"Lite 모드" 필요성 확인**: 마이크로 비즈니스에게는 핵심 SOP 3-5개(콘텐츠+SEO+소셜)만 실행하는 경량 모드가 실용적.
4. **네이버 쇼핑 SEO ≠ Google SEO**: 완전히 다른 알고리즘(키워드 적합성, 리뷰 수, 찜 수, 최근 판매량, 가격 경쟁력). 별도 SOP 필요.
5. **빈티지/구제 시장 데이터**: 한국 중고 의류 시장 약 5조 원, 연 30% 성장 (패션 전체 2.3% 대비 10배). 2027년 전체 의류의 24.3% 전망.

### [2026-03-21 / Sample Client E 재테스트 — 빈티지 가구, 신규 SOP 검증]

1. **신규 SOP 3개 모두 작동 확인**: one-item-inventory(일품 재고), naver-shopping-seo(네이버 쇼핑), visual-content-guide(비주얼) — 빈티지 가구에서 정상 작동.
2. **O2O 퍼널 패턴 발견**: 고가 일품(가구, 미술품 등)은 "온라인 발견 → 매장 방문 예약 → 현장 구매" 퍼널. 기존 온라인 전환 퍼널과 다름.
3. **업종별 비주얼 촬영 차이**: 가구(6샷: 전체/디테일/소재/스케일/스타일링/컨디션) vs 의류(착용컷/플랫레이/디테일) vs 용품(사용 시연/안전 인증)
4. **상세페이지 유연성 필요**: 24섹션 고정이 아닌, 업종별로 18-24섹션 유동적 구성. 가구는 "기능 Q&A" → "스토리 씬"으로 변형.
5. **Lite 모드 적합성 확인**: 마이크로 빈티지 가구점에 Lite 모드(5개 SOP) + 일품 재고 전략 조합이 최적.
6. **3개 클라이언트 업종 패턴 DB**:
   - 유아 의류(Sample Client C): 감성/디자인 → Social Channel A 핵심 → 시즌 캠페인
   - 유아 용품(Sample Client D): 안전/인증 → 멀티채널 → 성장단계 업그레이드
   - 빈티지 가구(Sample Client E): 희소성/큐레이션 → O2O 퍼널 → 실시간 콘텐츠

### [2026-07-02 / 런치팩 선형 체인 설계]

1. **시니어(50-60대)·비기술 클라이언트 = 선형 파이프라인이 정답**: 51개 분기 매핑은 전문가용. 신규·포괄 요청은 launch-pack 체인(상담 10문항 → 리서치 → 전략 → 채널믹스 → 후킹문안 → 브랜드킷 → 홈페이지 → 검증·리포트) 하나로 라우팅. 러너 스모크 테스트로 12스텝 선형 실행 + owner_gate 승인 정지 + 재개 완주 검증 완료.
2. **상담 질문 설계 원칙**: 전문용어(ICP/UVP/CTA) 발화 금지, 질문당 fallback 예시 필수, "모른다"는 정상 응답으로 처리(proof 없으면 "업계 자료로 대체" 명기, 지어내지 않음), 종료 전 요약 재확인 루프 필수.
3. **후킹문안은 독립 산출물**: 카피가 랜딩/SNS/광고에 흩어지면 사장님이 못 찾는다. 채널별 문안집(+어디에/왜 먹히는지 한 줄) 형태의 hooking-copy 플레이북으로 분리하니 재사용 가능해짐.

### [2026-07-02 / 09-local 모듈 설계]

1. **채널 우선순위 공식 (출처: docs/research/2026-07-02-local-sop-research.md)**: 전 업종 1순위 네이버 플레이스(검색 80%+) → 당근(동네)+카톡채널(단골 CRM, 도달 98.9%) → 음식점은 배민+쿠팡이츠 병행 트랙 → 구글은 관광 상권 조건부 → 오프라인은 5060 상권 보조. 5060 고객 상권이면 인스타 후순위.
2. **업종 3강 분기**: 50대+ 신규 창업 = 통신판매(콘텐츠·상세페이지 동선) / 한식음식점(플레이스·리뷰·배달앱) / 부동산중개(블로그·지역 키워드) — 채널믹스 전에 업종 분기 선행.
3. **0단계 = 무료 지원 + 사기 방어**: 돈 쓰기 전 카카오 단골 무상캐시 30만 원·소진공 온라인판로·지자체 중장년 지원 신청 안내 + 대행 사기 감별표 제공 — 시니어 클라이언트 신뢰 형성의 첫 단계 (공정위 기만영업 수사의뢰 18개 업체, 2026).

### [2026-07-02 / 광고·SEO·SNS 딥리서치 반영]

1. **SNS 산출물은 '공유 유발 설계'를 기획 단계에 명시**: 인스타 신규 도달 1등 신호 = 도달 대비 DM 공유 수 (Mosseri 공식, 2024-2026). 좋아요 KPI 폐기 → 공유·완주율·저장 KPI로 교체 완료 (social-post.md).
2. **유료 광고 SOP 공통 골격**: 계정 세팅 체크(전환 추적 필수) → 예산 구간별 구조안 → 소재 N컨셉 → 정책 사전 심사 → 주간 점검(끄기·증액 룰 수치화) → 사기 방어 1장. 네이버(파워링크)와 메타 모두 이 골격으로 작성 — 향후 광고 채널 추가 시 재사용.
3. **GEO 실증 공식**: 인용문 +41% / 통계 +31% / 출처 +30% / 키워드 스터핑 −10% (arXiv:2311.09735). 후발 사이트일수록 효과 큼(5위 +115%). llms.txt는 만들지 않는다(효과 근거 없음 — Ahrefs 97% 미열람). → prompts/shared/geo-checklist.md 성문화.
4. **벤치마크 정직 원칙**: 한국 공식 업종별 광고 벤치마크는 존재하지 않음 — 대행사 추정치는 [추정] 표기하고 "첫 30일 자체 데이터가 기준선"을 산출물에 명시한다.
