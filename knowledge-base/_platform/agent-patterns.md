# 에이전트 실행 패턴 (Platform-Level)

> 에이전트 실행 시 효과적이었던 패턴과 실패 패턴을 축적합니다.
> 에이전트는 작업 전 이 파일을 참조하여 검증된 패턴을 우선 적용합니다.
> Append-only. 항목당 최대 300자.

---

### [2026-03-21 / 검증된 성공 패턴]

1. **JSON 핸드오프 > 마크다운 전문 전달**: 에이전트 간 데이터 전달 시 JSON 핸드오프가 토큰 비용 절감 + 필드 정확도 향상. summary(3줄) + key_findings + data 구조가 효과적.
2. **자체 검증 체크리스트**: reviewer 전 자체 체크로 FAIL 재실행 비용 50% 이상 절감 기대.
3. **소스 신뢰도 3등급**: high/medium/low 태깅이 출처 품질 추적에 효과적. 소규모 브랜드 특례가 현실적으로 필수.
4. **B2C 플랫폼 선택 로직**: config.md의 B2B/B2C 비중으로 자동 분기하는 패턴이 모든 콘텐츠 SOP에 적용 가능.
5. **Strategy SOP 체인 실행**: GTM→포지셔닝→채널→퍼널→가격→캠페인 순서가 산출물 품질을 크게 높임. 이전 SOP 결과가 다음 입력이 됨.

### [2026-03-21 / 검증된 실패 패턴 (회피)]

1. **B2B 가정 기본값**: SOP/에이전트의 기본 가정이 B2B SaaS면 B2C 클라이언트에서 50%+ 항목이 부적합. 반드시 B2B/B2C 분기 필요.
2. **필수 입력 미확인 시 중단**: 예산/KPI가 없다고 SOP를 중단하면 아무것도 생산 못함. 비중(%)로 대체 + 워터마크가 더 실용적.
3. **3채널 고정 출력**: LinkedIn/Instagram/Twitter 고정은 B2C에서 비효율. 활성 채널 기반 동적 선택 필수.
4. **온보딩 시 config.md만 생성**: brand-guidelines, copy-patterns, pricing-rules 미생성 시 모든 후속 SOP에 연쇄 영향.

### [2026-03-21 / 업종별 특이사항]

1. **유아동복 D2C**: 인스타그램이 핵심 채널, 맘카페가 커뮤니티 중심, 시즌성 강함(S/S, F/W), 안전/인증이 신뢰 핵심, 감성 브랜딩 중요도 높음.

### [2026-03-21 / 2차 재테스트 — 신규 검증 패턴]

1. **B2C 플랫폼 분기 확인됨**: config.md의 B2B/B2C 비중 → B2C>50%이면 Instagram+네이버블로그 기본, LinkedIn 제외. 30개 산출물에서 일관 작동.
2. **예산 fallback 패턴 확인됨**: [미확인] 워터마크 + 비중(%)만 제시 패턴이 전략 6개 SOP 전체에서 정상 작동.
3. **200자 이메일 제한 확인됨**: 아웃바운드 3통 모두 200자 이하 (162/170/148자). 간결성이 오히려 품질 향상.
4. **GEO 스코어링 통합 확인됨**: SEO 감사에 GEO 8/30 포함. AI 인용 가능성, E-E-A-T, 크롤러 접근이 별도 카테고리로 분리되어 실행됨.
5. **고객 스토리 변형 확인됨**: B2C에서 SCSR 대신 5단계(고객소개→구매전고민→선택이유→사용경험→추천) 구조가 자연스럽게 작동.

### [2026-03-21 / 모윰 테스트 — 학습 루프 2회차]

**자기 개선 루프 성과: 헤이두두 교훈 → 모윰에 100% 자동 적용 → 재테스트 불필요**

1. **Platform KB 학습 루프 작동 확인**: 2번째 클라이언트에서 동일 실수 0건. 교훈 축적→참조→적용 사이클이 정상 작동.
2. **유아용품 B2C 패턴**: 젖병/빨대컵 등 기능성 제품은 안전/인증/소재가 카피 핵심. 의류(감성/디자인)와 다름. 업종 세분화 필요.
3. **멀티 마켓플레이스 브랜드**: 5개+ 채널 판매 시 가격 일관성 관리가 핵심 과제. 단일 채널 D2C와 전략이 다름.
4. **글로벌 진출 브랜드**: 해외 시장이 있으면 SEO에 hreflang, 다국어 콘텐츠, 현지화 전략이 추가로 필요.
5. **포지셔닝 축 패턴**: 유아 B2C 카테고리(의류+용품 모두)에서 "디자인 감성 x 소재 품질" 축이 범용적으로 작동. 업종별 범용 축 DB 구축 가능성.
6. **아웃바운드 200자 제한 재확인**: 모윰 183/173/145자 — 헤이두두 162/170/148자와 유사 범위. 200자 제한이 현실적으로 작동.

### [2026-03-21 / 업타운 빈티지 테스트 — 플랫폼 한계 패턴]

1. **플랫폼 호환성 매트릭스 발견**: 현재 SOP는 "자사몰 보유 + 대량생산 + 중소기업" 기준 설계. 이 3조건 중 하나라도 벗어나면 호환성 급락.
2. **비즈니스 유형별 호환 점수**: D2C 자사몰(9/10), 마켓플레이스 네이티브(4/10), 일품 재고(3/10), 마이크로 1-3인(3/10)
3. **"Lite 모드" 필요성 확인**: 마이크로 비즈니스에게는 핵심 SOP 3-5개(콘텐츠+SEO+소셜)만 실행하는 경량 모드가 실용적.
4. **네이버 쇼핑 SEO ≠ Google SEO**: 완전히 다른 알고리즘(키워드 적합성, 리뷰 수, 찜 수, 최근 판매량, 가격 경쟁력). 별도 SOP 필요.
5. **빈티지/구제 시장 데이터**: 한국 중고 의류 시장 약 5조 원, 연 30% 성장 (패션 전체 2.3% 대비 10배). 2027년 전체 의류의 24.3% 전망.
