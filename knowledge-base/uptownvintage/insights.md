# 업타운 빈티지 -- 리서치 인사이트

> 날짜별 인사이트 축적. Append-only. 항목당 최대 500자.

---

### [2026-03-21 / 온보딩 리서치]

1. **한국 빈티지 가구 시장 급성장**: 2020년대 중반부터 빈티지 가구 딜러 급증. 미드센츄리 모던 중심으로 시장 형성. 전체 가구 시장(30조 원) 내 빈티지 세그먼트 성장 속도가 가장 빠름. 업사이클링/지속가능 소비 트렌드와 결합하여 구조적 성장 중.

2. **Instagram이 핵심 발견 채널**: 빈티지 가구 딜러 생태계가 Instagram 중심으로 형성. 소비자는 딜러를 팔로우해두고 신규 입고를 기다리는 패턴. "팔로우 -> 발견 -> 문의/구매" 퍼널이 자연스럽게 작동.

3. **일품 재고 비즈니스의 마케팅 특수성**: 표준 가격표/재입고/리마케팅이 불가능. 각 상품이 하나의 캠페인. FOMO(놓치면 끝)가 가장 강력한 전환 트리거. 상품 페이지가 판매 후 소멸하므로 "SOLD" 콘텐츠가 사회적 증거 + 긴급성 동시 전달.

4. **ICP: 공간 큐레이터 (28-45세 인테리어 애호가)**: 대량생산 가구에 만족하지 못하고 개성 있는 공간을 원하는 소비자. 미드센츄리 모던/북유럽 빈티지에 관심. 오늘의집, Instagram, Pinterest에서 정보 탐색. 매장 방문(실물 확인)이 구매 전환의 핵심.

5. **멀티채널 선순환 가능성**: 오프라인 매장(체험) -> Instagram(팔로우/재방문) -> 스마트스토어(구매) 선순환 구조 구축 가능. 링크트리로 채널 허브 이미 확보.
