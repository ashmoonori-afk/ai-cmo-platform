# 교훈 기록 -- 업타운 빈티지 (Uptown Vintage)

> 마케팅 전략 실행에서 얻은 성공/실패 교훈을 누적 기록합니다.

---

### [2026-03-21 / 클라이언트 온보딩]

**온보딩 과정 교훈:**
1. 네이버 스마트스토어 네이티브 브랜드는 자사몰 중심 SOP와 근본적 차이. GA4 감사, 사이트 SEO, 랜딩페이지 디자인 등 자사몰 전제 SOP가 적용 불가. 네이버 쇼핑 SEO/스마트스토어 최적화 SOP 별도 필요.
2. 빈티지/구제 비즈니스의 1점 한정 재고 특성은 기존 플랫폼의 재고 관리, 재주문 마케팅, 동일 상품 A/B 테스트 등 SOP를 무력화. "매 상품 등록이 곧 신제품 런칭"이라는 사고 전환 필요.
3. 마이크로 비즈니스(1-3인)는 11개 에이전트의 풀 실행이 비현실적. 3-5개 핵심 SOP만 선별 실행하는 "라이트 모드"가 필요.
4. 빈티지 의류의 가격 책정은 "정가 대비 할인" 프레임이 아닌 "아이템별 고유 가치 평가" 프레임. pricing-strategy SOP에 빈티지/일품 가격 모델 추가 검토 필요.
5. 스마트스토어 직접 웹 크롤링이 불가한 경우(JS 렌더링, 봇 차단), 플레이라이트 기반 접근 또는 네이버 검색 API 활용 등 대안 필요. WebFetch fallback 패턴을 마켓플레이스 환경에 맞게 확장해야 함.
