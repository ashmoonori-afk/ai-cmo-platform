# GEO 체크리스트 (AI 검색 인용 최적화)

> seo-specialist·copywriter가 검색성 콘텐츠 제작 시, reviewer가 검증 시 적용.
> 근거: `docs/research/2026-07-02-seo-geo-research.md` (GEO 논문 arXiv:2311.09735 KDD 2024 실증 수치 포함)

## 왜 하는가

- 네이버 AI 브리핑이 통합검색 쿼리 20% 돌파 (2025-12), AI Overviews 노출 시 1위 CTR −58% (Ahrefs)
- AI 검색발 방문자는 오가닉 대비 전환 4.4배 (Semrush 2026) — 트래픽은 작아도 질이 높음
- 후발 사이트에 유리: 검색 5위 사이트가 출처 인용 적용 시 가시성 +115% (GEO 논문 — "민주화 효과")

## 10항목 체크리스트

| # | 항목 | 근거 (개선율) |
|---|------|-------------|
| 1 | 첫 H2 이전에 **직답 문단** (40~60단어, 질문에 대한 정의·결론 선제시) | AI 발췌 단위 |
| 2 | **출처 있는 통계 문장 2개 이상** ("○○에 따르면 N%") | +31% (논문) |
| 3 | **인용문 1개 이상** (전문가·공식 발언 직접 인용) | +41% (논문 — 최고 효과) |
| 4 | **출처 링크 명시** (신뢰 가능한 원출처) | +30% (논문) |

| 6 | 최신성: 갱신일 표기 + 상위 트래픽 글 분기별 리프레시 | HubSpot 리프레시 +106% |
| 7 | Q&A·표·목록 구조 (스캔 가능한 단위) | AI 브리핑 인용 관측 [추정] |
| 8 | 저자·브랜드 정보 명시 (E-E-A-T) | 구글 2024 코어 프레임 |
| 9 | **키워드 스터핑 금지** | GEO에서 −10% (논문) + 구글 스팸 정책 |
| 10 | AI 크롤러 허용 확인 (아래 스니펫) | 차단 시 AI 검색 노출 상실 |

## AI 크롤러 robots.txt (소상공인 기본값 = 전부 허용)

```
# AI 검색 노출 허용 (기본 권장)
User-agent: OAI-SearchBot
Allow: /
User-agent: PerplexityBot
Allow: /

# 모델 학습만 거부하고 싶을 때 (선택 — 검색 노출과 별개)
# User-agent: GPTBot
# Disallow: /
# User-agent: Google-Extended
# Disallow: /
```

- Google-Extended 차단은 **구글 검색 랭킹에 영향 없음** (구글 공식)
- Perplexity-User(사용자 실시간 요청)는 "일반적으로 robots.txt 무시" (Perplexity 공식) — 차단 불가 전제
- **llms.txt는 만들지 않는다**: 효과 근거 없음 (구글 "어떤 AI도 사용 안 함"; Ahrefs 실측 97% 미열람, 2026)
- 스키마는 위생 요소만 (Organization/FAQ/Article 3종) — AI 인용 직접 효과는 근거 상충
