## 1. 헤드라인 패턴

빈티지 가구의 특성(일품 재고, 시대적 가치, 공간 변화)을 반영한 헤드라인 구조.

### 패턴 A: 시대 + 아이템 + 감성
> "{시대} {원산지}에서 온 {아이템}. {감성 한 줄}."
- 예: "1960년대 덴마크에서 온 티크 사이드보드. 반세기를 견딘 균형의 미학."
- 예: "1970년대 독일에서 건너온 월넛 데스크. 시간이 깊게 만든 나무결."

### 패턴 B: 희소성 강조 (FOMO)
> "단 하나뿐인 {아이템} -- {차별화 포인트}"
- 예: "단 하나뿐인 Wegner 라운지체어 -- 오리지널 패브릭 그대로"
- 예: "이번이 마지막입니다. {디자이너} {아이템}, 이 컨디션은 다시 없습니다."

### 패턴 C: 공간 변화 약속
> "이 {아이템} 하나로, {공간}이 달라집니다"
- 예: "이 플로어램프 하나로, 거실의 분위기가 완전히 달라집니다."
- 예: "사이드보드 하나가 만드는 다이닝 공간의 격."

### 패턴 D: 질문형 (관심 유도)
> "{인테리어 고민}? {해결책으로서의 가구}"
- 예: "거실이 밋밋하다고 느끼셨나요? 50년 된 가구 하나가 답입니다."

### 패턴 E: 슬로건 변형
> "과거의 멋을 현대로 -- {구체적 상황}"
- 예: "과거의 멋을 현대로 -- 이번 주 새로 도착한 셀렉션을 소개합니다."

## 2. CTA 패턴

빈티지 가구 구매 특성에 맞는 행동 유도 문구. 일품 재고의 긴급성 + 오프라인 체험 유도가 핵심.

| CTA 유형 | 문구 | 사용 맥락 |
|---------|------|---------|
| 매장 방문 유도 | "매장에서 직접 만나보세요" | 오프라인 체험 강조 시 |
| 예약 유도 | "네이버 예약으로 방문 일정 잡기" | 매장 방문 전환 |
| 긴급성 | "지금 놓치면 다시 만날 수 없습니다" | 일품 재고 FOMO |
| 스마트스토어 | "스마트스토어에서 자세히 보기" | 온라인 상세 확인 |
| DM 문의 | "DM으로 문의해 주세요" | Social Channel A 전환 |
| 새 입고 알림 | "새로운 셀렉션, 가장 먼저 만나보세요" | 팔로우/알림 설정 유도 |
| 가벼운 탐색 | "이번 주 셀렉션 구경하기" | 부담 없는 브라우징 유도 |

## 3. 제품 설명 패턴 (가구/소품)

빈티지 가구 상품 설명은 아래 구조를 따른다.

### 구조: 7요소 상품 설명

```
1. [아이템명] -- 한 줄 감성 카피
2. 기본 정보: 디자이너(알 수 있는 경우) / 제조사 / 시대 / 원산지
3. 소재 & 마감: 주요 소재, 마감 방식, 특이사항
4. 치수: 가로 x 세로 x 높이 (cm)
5. 컨디션: 전체 상태 설명 + 사용감/보수 이력
6. 스타일링 제안: 어떤 공간에 어울리는지 구체적 제안
7. 가격 + CTA
```

**예시**:
```
[덴마크 티크 사이드보드] -- 반세기를 견딘 균형의 미학

- 시대: 1960년대 추정
- 원산지: 덴마크
- 소재: 티크 무늬목 + 솔리드 티크 다리
- 치수: W150 x D45 x H75 cm
- 컨디션: 양호. 상판에 미세한 사용 흔적 있으나 구조적으로 견고. 서랍 슬라이딩 원활.
- 스타일링: 거실 TV 콘솔로, 혹은 다이닝 공간의 식기 수납으로. 화이트 벽 + 그린 식물과 조합하면 미드센츄리 무드가 완성됩니다.
- 가격: ₩OOO,OOO
- 매장에서 직접 확인하실 수 있습니다. (네이버 예약)
```

{감성 설명 2-3줄}

{치수 정보}
{가격 또는 "DM 문의"}

#빈티지가구 #미드센츄리모던 #SampleClientE #빈티지인테리어 #vintageinterior
```

### 4-2. 스타일링/공간 연출
```
{공간 사진과 함께}

{이 공간이 어떻게 만들어졌는지 짧은 이야기}

과거의 멋을 현대로.
매장에서 직접 만나보세요.

#빈티지인테리어 #공간스타일링 #SampleClientE #vintagehomedecor

{간단한 아이템 설명 1줄}

## 5. 금지 패턴

| 패턴 | 이유 |
|------|------|
| "최저가 보장" / "가격 비교" | 일품 재고는 가격 비교 대상 없음. 가치 기반 판매 |

| "중고 가구 세일" | 브랜드 포지셔닝 훼손 |
| 과도한 이모지 남발 | 큐레이토리얼 톤에 부적합 |

| "Social Channel A 감성" (직접 사용) | 빈티지 자체의 가치를 SNS 트렌드로 환원하는 인상 |
