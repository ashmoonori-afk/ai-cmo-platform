# 가격 정책 — Sample Client B

## 사업 구조 개요
Sample Client B의 가격 정책은 크게 세 영역으로 구분된다.
1. **컨세션 식음료 직매출** — 빈소 가족·조문객 대상 음식·제사상 판매
2. **위탁 운영 수수료** — 외부 장례식장으로부터 받는 운영 위탁 대가
3. **B2B SaaS / 플랫폼** — 장례식장 관리 솔루션 구독 및 플랫폼 수수료

---

## 1. 식음료 패키지 가격 체계 (빈소 대상 직매출)

### 카테고리별 품목 구성
| 카테고리 | 대표 품목 | 비고 |
|---------|---------|------|
| 식사류 | 쌀밥, 국류 등 | 포맷 A Branch A 기준 10- 코드 |
| 육류 | 삼겹수육 등 | 20- 코드 |
| 전류 | 각종 전 | 30- 코드 |
| 일품요리류 | 단품 요리 | 40- 코드 |
| 반찬류 | 각종 반찬 | 50- 코드 |
| 양념류 | 소스류 | 60- 코드 |
| 떡류 | 각종 떡 | 70- 코드 |
| 제사상 | 제사 세트 | 80- 코드 |
| 안주류 | 음주 안주 | Branch B 지점 중심 |
| 후식류 | 과일, 디저트 등 | Branch B 지점 중심 |

### 패키지 구성 방향
| 패키지 유형 | 구성 원칙 | 타겟 상황 |
|-----------|---------|---------|
| 기본형 | 식사류 + 반찬류 중심, 최소 구성 | 소규모 가족장, 예산 우선 |
| 표준형 | 식사 + 반찬 + 안주 + 제사상 기본 구성 | 일반 빈장 (표준 수요) |
| 프리미엄형 | 전 카테고리 포함 + 대용량 세트 | 대규모 빈장, 조문객 다수 |
| 종교별 맞춤 | 불교/기독교/천주교/무종교 맞춤 구성 | 종교적 금기 반영 필요 시 |
| 제사상 단독 | 제사상 세트만 별도 주문 | 49재, 탈상 등 추가 제사 |

### 가격 설정 원칙
- 단가는 지점별 현지 물가 및 경쟁 장례식장 수준을 반영하여 개별 설정
- 세트 할인은 매출 데이터 기반으로 할인율 조정 (음수 금액 항목 존재 — 포맷 A Branch A 참조)
- 수량이 소수점인 품목(싯가 품목)은 별도 단가 협의
- 지역별 표준 단가 범위: 미확인 (데이터 분석 후 업데이트 필요)

---

## 2. 위탁 운영 수수료 체계

### 구조 개요
- 외부 장례식장이 컨세션 운영을 Sample Client B에 위탁할 때 적용
- 수수료 구조는 매출 연동형 또는 고정 운영비 방식으로 협의
- 현재 5개 위탁 지점 운영 중

### 가격 커뮤니케이션 규칙
- 위탁 수수료율: **문의 시 협의** (공개 미확인)
- 영업 제안 단계에서는 현재 운영 지점 실적 데이터를 근거로 수익성 시뮬레이션 제공
- 계약 조건은 개별 장례식장 규모·빈장 건수·메뉴 구성 복잡도에 따라 차등 적용

---

## 3. B2B SaaS / 플랫폼 가격 체계

### 현황
- 장례식장 관리 SaaS: 가격 미확인 (50개 이상 장례식장 도입 실적 있음)
- 모바일 부고 서비스: 건당 과금 방식 추정 (연간 10,000건 이상 사용 실적)
- 플랫폼 중개 수수료: 미확인 (향후 IR 자료 기반 업데이트 필요)

## 4. 할인 정책
| 조건 | 할인 방식 | 비고 |
|------|---------|------|
| 세트 메뉴 구성 | 개별 품목 합산 대비 할인 적용 | 음수 금액 항목으로 반영 (포맷 A) |
| 다량 주문 (대규모 빈장) | 볼륨 기반 협의 할인 | 조문객 수 기준 |
| 연간 위탁 계약 | 협의 | 장기 파트너십 우대 |
| 신규 위탁 지점 도입 | 초기 지원 패키지 별도 협의 | 시범 운영 기간 적용 가능 |

## 5. 경쟁사 가격 비교
| 경쟁 유형 | 가격 범위 | 대비 포지셔닝 |
|---------|---------|------------|
| 장례식장 자체 운영 (직고용) | 인건비·식재료 직접 부담 | 위탁 전환 시 비용 구조 개선 가능 |
| 지역 외식 위탁업체 | 미확인 | 장례 특화 전문성으로 차별화 |
| 대형 상조회사 식음료 | 미확인 | 지역 맞춤 대응 속도로 차별화 |

> **주의**: 경쟁사 가격 비교 시 직접 비하 금지. Sample Client B의 전문성·실적·데이터 기반 운영을 중심으로 포지셔닝.

## 6. 가격 커뮤니케이션 규칙
- **공개 범위**: 식음료 단품 가격은 빈소 현장 메뉴판 공개 / 위탁 수수료는 문의 시 안내
- **할인 권한**: 현장 운영자는 세트 구성 내 조정 가능 / 위탁 조건 변경은 본사 승인 필요
- **금지**: 경쟁사 가격 직접 비교 비하 금지 / 원가율 공개 금지
- **IR 커뮤니케이션**: 투자자 대상으로는 지점별 매출 규모 및 성장률 중심 제시
