# 브랜드 가이드라인 — Sample Client B

## 브랜드 미션
장례 현장의 식음료 운영을 전문화하여, 슬픔의 자리에서도 품격 있는 식사와 정성스러운 제사상을 제공한다.

## 톤앤매너

### 브랜드 성격
- 신뢰감 있고 절제된 전문가 — 감정을 앞세우지 않고, 안정감과 배려를 행동으로 보여준다
- 진중하면서도 실용적 — 슬픔의 자리에 어울리는 무게감을 유지하되, 과도한 감성 표현은 지양
- B2B 파트너십 중심 — 장례식장 운영사를 대상으로 할 때는 데이터와 실적을 중심으로 말한다

### 문체 규칙
- **격식 수준**: 격식체 (B2B 커뮤니케이션) / 반격식체 (장례식장 운영자 대상 안내 자료)
- **인칭**: 저희 (B2B 외부 커뮤니케이션) / 우리 (내부 전략 문서)
- **문장 길이**: 간결하고 명확하게. 불필요한 수식어 배제. 숫자와 근거로 말한다.

### 선호 표현
| 사용 O | 사용 X |
|--------|--------|
| 데이터 기반 / 실적 기반 | 최고 / 최상 (근거 없는 최상급) |
| 전문 운영 / 전문 인력 | 업계 최고 / 압도적 (과장) |
| 안정적 운영 / 지속 가능한 수익 | 폭발적 성장 / 혁신적 |
| 정성스러운 / 품격 있는 | 화려한 / 럭셔리 |
| 빈소별 맞춤 / 지역 특화 | 대량 공급 / 규격화 |
| 운영 효율화 / 비용 절감 | 저렴한 / 싼 |
| ~을 지원합니다 / ~을 함께합니다 | ~해드립니다 (지나치게 낮은 자세) |

### 금지 표현
- 사망, 죽음, 시신 등 직접적인 단어를 마케팅 카피에 사용하지 않는다
- 슬픔·눈물 등 감정을 자극하는 표현으로 영업하지 않는다
- 경쟁사 비하 또는 직접 비교 금지
- 근거 없는 최상급 표현 (최고, 최대, 유일 등) 금지
- 장례 현장의 엄숙함을 해치는 경쾌하거나 가벼운 표현 금지

## 핵심 가치 제안
- **메인 메시지**: 장례 현장의 식음료 운영, 전문가에게 맡기십시오
- **서브 메시지 1**: 9개 지점 운영 데이터로 검증된 메뉴 구성과 객단가 최적화
- **서브 메시지 2**: 지역 특성과 종교별 관행을 반영한 빈소 맞춤 제사상
- **서브 메시지 3**: 운영 리스크는 줄이고, 본업 집중도는 높이는 위탁 파트너십

## 채널별 커뮤니케이션 방향

### B2B 영업 자료 (제안서, 피칭덱)
- 실적 수치 중심: 운영 지점 수, 객단가, 빈장 건수, 매출 증감
- 감성보다 ROI — "위탁 후 운영 비용 XX% 절감 사례" 형식
- 도표와 데이터 시각화 적극 활용

### IR 자료 (투자자 대상)
- 시장 규모와 성장 가능성을 수치로 제시
- 플랫폼 확장 로드맵 명확히
- 창업팀 역량과 기존 트랙레코드 강조

### 운영 안내 자료 (장례식장 스태프 대상)
- 간결하고 명확한 지침 형식
- 존댓말 유지, 친절하지만 불필요한 수사 배제
