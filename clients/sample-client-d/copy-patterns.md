## 1. 헤드라인 패턴

공개 자료에서 추출한 대표 헤드라인 구조:

| # | 패턴 | 예시 | 특징 |
|---|------|------|------|
| 1 | **기능+감성 결합형** | "아이의 모든 순간을 함께하는 프리미엄 유아용품" | 브랜드 철학+카테고리 |
| 2 | **제품명+소재 키워드** | "3세대 NEW PPSU 노꼭지 올인원 젖병" | 기술 스펙 명확 전달 |
| 3 | **에디션+시리즈형** | "270ml 트리플팩 래빗 에디션 (핑크)" | 컬렉터블 감성 + 색상 명시 |
| 4 | **인증/수상 강조형** | "4년 연속 한국소비자만족지수 1위" | 신뢰도 중심 |
| 5 | **성장 단계 안내형** | "젖병에서 빨대컵까지, 올인원 시스템" | 부모의 실용적 니즈 대응 |

### 헤드라인 작성 규칙
- 한글 기준 25자 이내, 영문 기준 10단어 이내
- 안전성 키워드(BPA-free, PPSU, 마이크로플라스틱 프리) 1개 이상 포함 권장
- 감탄사, 느낌표 최소 사용 (프리미엄 톤 유지)
- 에디션명 포함 시 색상/동물 모티브 함께 표기

## 2. CTA(Call-to-Action) 패턴

공개 자료 및 자사몰에서 확인된 행동 유도 문구:

| CTA | 사용 위치 | 톤 |
|-----|---------|-----|
| "자세히 보기" | 제품 카테고리 | 부드러운 탐색 유도 |
| "장바구니 담기" | 제품 상세 | 기본 구매 유도 |
| "세트로 만나보기" | 번들 상품 | 객단가 상승 유도 |
| "성장 단계별 추천" | 가이드 페이지 | 정보 제공형 |

### CTA 작성 규칙
- 한글 CTA 중심 (자사몰 기준)
- "만나보세요", "확인해보세요" 등 부드러운 권유형 선호
- 금지: "지금 바로!", "서두르세요!", "놓치지 마세요!" 등 긴급성 강조 표현
- 세트/번들 상품 CTA에서 절약 금액보다 "함께 쓰면 더 좋은" 가치 프레이밍

## 3. 제품 설명 패턴

자사몰 및 Global Marketplace A 제품 페이지에서 반복되는 설명 구조:

### 기본 구조
```
[제품명 + 에디션] -- [핵심 기능/소재] -- [안전 인증] -- [사이즈/용량] -- [호환성]
```

### 설명 레이어
1. **안전 레이어**: 소재 안전성 강조 (1-2문장)
   - 예: "BPA-free, 마이크로플라스틱 프리 PPSU 소재로 안심하고 사용하세요."
2. **기능 레이어**: 올인원 시스템, 소독 방법, 호환 액세서리 (목록형)
   - 예: "열탕/스팀/전자레인지/식기세척기 소독 가능"
3. **디자인 레이어**: 에디션 디자인, 컬러, 감성 (1문장)
   - 예: "사랑스러운 래빗 에디션으로 수유 시간을 더 특별하게."
4. **성장 레이어**: 단계별 전환 안내 (해당 시)
   - 예: "젖꼭지를 빨대로 교체하면 빨대컵으로 변신! 아이 성장에 맞춰 함께합니다."

### 제품 설명 규칙
- 첫 문장은 반드시 안전/소재 관련 (부모의 핵심 관심사)
- 전문 인증(소아치과학회 추천, Good Design Award)은 별도 배지/아이콘 처리
- 용량/사이즈는 표로 정리 (비교 용이성)
- 에디션별 차이점 명확 표기

## 4. SNS 카피 패턴

Social Channel A 공식 페이지 기반 추정 패턴:

### 기본 구조
```
[부모 공감 한 줄] + [제품 소개 2-3줄] + [인증/어워드 1줄] + [해시태그]
```

### 예시 패턴
```
아이가 처음 만나는 젖병, 안전이 먼저입니다.

Sample Client D 3세대 PPSU 올인원 젖병은
BPA-free, 마이크로플라스틱 프리 소재로
아이의 건강한 첫 시작을 함께합니다.

4년 연속 한국소비자만족지수 1위

#SampleClientD #SampleClientD #프리미엄젖병 #PPSU젖병 #육아용품 #출산준비
```

### SNS 작성 규칙
- 첫 줄은 부모 공감/안전 메시지 (스크롤 멈춤 유도)
- 본문 3-4줄 이내 (간결)
- 인증/수상 실적 1개 이상 포함 (신뢰도)
- 해시태그: 브랜드 태그 2개(#SampleClientD #SampleClientD) + 카테고리 태그 3-5개
- 이모지 사용: 최소한으로 (프리미엄 톤 유지)
- 에디션 출시 포스트: 컬러/디자인 비주얼 중심

## 5. 금지 패턴

브랜드 톤앤매너에 맞지 않는 카피 스타일:

| 금지 패턴 | 예시 | 대안 |
|---------|------|------|
| **공포 마케팅** | "플라스틱 젖병의 숨겨진 위험!" | "안심할 수 있는 PPSU 소재를 선택하세요" |
| **긴급성 강조** | "오늘만! 한정 수량!" | "새로운 에디션을 만나보세요" |
| **과장형 수식** | "역대급! 혁신적! 미친 퀄리티!" | "사려 깊게 디자인한 프리미엄 제품" |
| **비교 공격** | "다른 젖병보다 훨씬 안전한" | (비교 자체를 하지 않음, 자체 인증으로 증명) |
| **죄책감 유발** | "좋은 엄마라면 이건 필수" | "아이의 안전한 첫 시작을 위해" |
| **과도한 이모지** | 문장마다 이모지 3개 이상 | 이모지 없이 또는 최소 1-2개 |
| **유행어/밈** | "갓벽! 레전드! 찐이야!" | 정제되고 신뢰감 있는 표현 |
