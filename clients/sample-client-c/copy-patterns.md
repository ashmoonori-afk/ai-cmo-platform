## 1. 헤드라인 패턴

사이트에서 추출한 대표 헤드라인 구조:

| # | 패턴 | 예시 | 특징 |
|---|------|------|------|
| 1 | **영문 감성 문장형** | "Made for little lives and the homes they grow in" | 브랜드 철학을 시적으로 전달 |
| 2 | **제품명 + 시즌 키워드** | "Comode Summer Baby Set" | 간결하게 제품+시즌 조합 |
| 3 | **감정 유발 짧은 문장** | "The smallest moments become the most beautiful memories" | 부모 감성 자극, 제품 직접 언급 없음 |
| 4 | **소재/감촉 중심** | "Soft, simple pieces that feel like everyday comfort" | 촉각적 경험 강조 |
| 5 | **컬렉션 이름형** | "Comode Collection" | 단순 명사구, 미니멀 |

### 헤드라인 작성 규칙
- 15단어 이내 (영문 기준) / 20자 이내 (한글 기준)
- 감탄사, 느낌표 사용 자제
- 형용사는 촉각/감각 관련 위주 (soft, gentle, cozy, warm)
- 상업적 수식어 배제 (최고, 인기, 핫한 등)

## 2. CTA(Call-to-Action) 패턴

사이트에서 확인된 행동 유도 문구:

| CTA | 사용 위치 | 톤 |
|-----|---------|-----|
| "SHOP NOW" | 메인 배너 | 직접적이지만 간결 |
| "VIEW MORE" | 컬렉션 페이지 | 탐색 유도 |
| "ADD TO CART" | 제품 상세 | 기본 구매 유도 |
| "OUR STORY" | 네비게이션 | 브랜드 스토리 유도 |

### CTA 작성 규칙
- 영문 대문자 스타일 유지 (브랜드 비주얼 일관성)
- 한글 CTA 사용 시: "자세히 보기", "장바구니 담기" 등 부드러운 표현
- 금지: "지금 바로!", "서두르세요!", "놓치지 마세요!" 등 긴급성 강조 표현

## 3. 제품 설명 패턴

사이트 제품 페이지에서 반복되는 설명 구조:

### 기본 구조
```
[제품명 (영문)] -- [소재/촉감 설명] -- [디자인 의도] -- [착용 연령/사이즈]
```

### 설명 레이어
1. **감성 레이어**: 제품이 주는 느낌/분위기 (1문장)
   - 예: "포근한 오후, 아이와 함께하는 조용한 시간을 위한 세트"
2. **기능 레이어**: 소재, 사이즈, 세탁 방법 (목록형)
   - 예: "100% 코튼 / 3-36개월 / 손세탁 권장"
3. **스타일 레이어**: 코디 제안 또는 활용 팁 (선택)
   - 예: "같은 라인의 보넷과 함께 스타일링하면 더욱 사랑스러워요"

### 제품 설명 규칙
- 첫 문장은 반드시 감성적 묘사 (기능 설명으로 시작하지 않음)
- 소재 설명 시 "부드러운", "자연스러운", "편안한" 수식어 활용
- 가격 강조/할인율 강조 최소화
- 사이즈 가이드는 별도 섹션으로 분리

## 4. SNS 카피 패턴

Social Channel A(@sample_client_c_official) 기반 추정 패턴:

### 기본 구조
```
[감성 영문 한 줄] + [한글 제품/시즌 소개 2-3줄] + [해시태그]
```

### 예시 패턴
```
Slow mornings, soft fabrics.

코모드 썸머 컬렉션이 도착했어요.
아이의 여름을 편안하게, 예쁘게.

#SampleClientC #SampleClientC #베이비룩 #유아복 #슬로우리빙
```

### SNS 작성 규칙
- 영문 캡션은 1줄 이내 (감성 키워드 중심)
- 한글 본문은 3줄 이내 (간결)
- 해시태그: 브랜드 태그 2개 + 카테고리 태그 3-5개
- 이모지 사용: 최소한으로 (과도한 이모지는 브랜드 톤과 불일치)
- 댓글/DM 응대: 따뜻하고 친근한 반말 또는 반존칭

## 5. 금지 패턴

브랜드 톤앤매너에 맞지 않는 카피 스타일:

| 금지 패턴 | 예시 | 대안 |
|---------|------|------|
| **긴급성 강조** | "오늘만! 한정 수량!" | "새로운 컬렉션을 만나보세요" |
| **과장형 수식** | "역대급 퀄리티! 미친 가성비!" | "정성을 담아 만든 소재" |
| **비교 공격** | "다른 브랜드보다 훨씬 좋은" | (비교 자체를 하지 않음) |
| **죄책감 유발** | "좋은 엄마라면 이건 필수" | "아이와 함께하는 편안한 순간" |
| **유행어/밈** | "갓벽! 레전드! 찐이야!" | 시적이고 절제된 표현 |
| **과도한 이모지** | 문장마다 이모지 삽입 | 이모지 없이 또는 최소 1-2개 |
