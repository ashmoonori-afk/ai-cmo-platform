## 1. 헤드라인 패턴 (상품명/타이틀)

빈티지 패션 스마트스토어의 상품명은 곧 검색 키워드이자 헤드라인이다.

| # | 패턴 | 예시 | 특징 |
|---|------|------|------|
| 1 | **[브랜드] + [아이템] + [연대] + [특징]** | "Levi's 501 빈티지 데님 90s USA 워싱" | 검색 최적화 + 핵심 정보 전달 |
| 2 | **[아이템 유형] + [소재/패턴] + [컬러] + [사이즈]** | "빈티지 울 체크 오버사이즈 자켓 브라운 L" | 필터링 용이, 범용적 |
| 3 | **[스토리형] + [아이템]** | "80년대 미국 캠퍼스에서 온 레터맨 자켓" | 감성적, SNS 공유 유도 |
| 4 | **[컨디션 강조] + [아이템]** | "상태 A급 | 빈티지 리바이스 트러커 자켓" | 구매 불안 해소 |
| 5 | **[한정/희소] + [아이템]** | "단 한 벌 | 70s 빈티지 플로럴 원피스" | 희소성 + 구매 긴급성 |

### 상품명 작성 규칙 [추정]
- 네이버 쇼핑 검색 알고리즘 고려: 핵심 키워드를 앞쪽에 배치
- 50자 내외 권장 (네이버 스마트스토어 노출 최적)
- 브랜드명 있으면 반드시 포함 (검색 유입)
- 사이즈/컬러는 상품명 또는 옵션으로 분리

## 2. CTA(Call-to-Action) 패턴

스마트스토어 환경에서의 행동 유도 문구:

| CTA | 사용 위치 | 톤 |
|-----|---------|-----|
| "지금 바로 구매하기" | 상품 상세페이지 | 직접적 (스마트스토어 기본 UI) |
| "찜하기 눌러두세요" | 상세페이지 하단/SNS | 부드러운 관여 유도 |
| "새 상품 알림 받기" | 스토어 팔로우 유도 | 재방문 유도 |
| "스토어 소식 받기" | SNS/스토어 프로필 | 구독 유도 |
| "이 아이템 놓치지 마세요" | 신상품 업로드 알림 | 희소성 강조 (빈티지 특성상 적합) |

### CTA 작성 규칙 [추정]
- 스마트스토어 UI 제약 내에서 최적화 (상세페이지 이미지 내 CTA 삽입)
- "찜"과 "소식받기"가 스마트스토어 생태계 핵심 CTA
- 빈티지 특성상 "한 벌뿐" 긴급성 CTA는 과장이 아닌 사실이므로 적극 활용 가능
- "선착순/한정수량" 표현 -- 빈티지는 실제로 1점이므로 허용

## 3. 제품 설명 패턴 (상세페이지)

빈티지 의류 상세페이지의 반복 구조:

### 기본 구조
```
[상품 대표 이미지 (2-3컷)]
→ [아이템 소개 1줄 (스토리/감성)]
→ [상세 스펙 테이블 (브랜드/연대/소재/사이즈/컨디션)]
→ [컨디션 상세 설명 + 하자 고지]
→ [실측 사이즈 (cm 단위)]
→ [착용 참고 사항]
→ [배송/교환/반품 안내]
```

### 설명 레이어
1. **스토리 레이어**: 아이템의 출처/연대/문화적 맥락 (1-2문장)
   - 예: "90년대 미국 워크웨어의 정수. 세월이 만든 자연스러운 워싱감이 매력적인 카하트 자켓입니다."
2. **스펙 레이어**: 브랜드, 소재, 사이즈, 원산지, 연대 (목록형)
   - 예: "브랜드: Carhartt / 소재: 100% Cotton Duck / 사이즈: XL / 원산지: USA / 연대: 90s"
3. **컨디션 레이어**: 상태 등급 + 하자 상세 (빈티지 핵심)
   - 예: "컨디션: A급 (하자 없음, 약간의 자연스러운 빈티지 사용감)"
4. **사이즈 레이어**: 실측 사이즈 (cm)
   - 예: "어깨 52 / 가슴 62 / 소매 63 / 총장 72 (단위: cm, 오차 1-2cm)"

### 제품 설명 규칙 [추정]
- 컨디션 설명은 과장 없이 솔직하게 (신뢰가 재구매의 핵심)
- 하자가 있으면 반드시 사진 + 텍스트로 고지
- 실측 사이즈는 필수 (빈티지는 표기 사이즈와 실측이 다를 수 있음)
- 소재 설명 시 촉감/두께감 언급 (온라인 구매 불안 해소)

## 4. SNS 카피 패턴

인스타그램 기반 추정 패턴 [추정]:

### 기본 구조
```
[아이템 사진 (착용 또는 행거)]
[스토리 1줄 + 아이템 소개]
[사이즈/가격 간략 정보]
[구매 링크 안내 (프로필 링크 or 스마트스토어)]
[해시태그]
```

### 예시 패턴 [추정]
```
80년대 미국에서 건너온 빈티지 스타디움 자켓.
이 패치워크 디테일, 실물이 더 예뻐요.

사이즈 L / 상태 A급
스마트스토어에서 만나보세요 (프로필 링크)

#업타운빈티지 #빈티지자켓 #구제 #빈티지패션 #vintagekorea
```

### SNS 작성 규칙 [추정]
- 아이템당 1포스트 (빈티지는 개별 아이템이 콘텐츠)
- 스토리/릴스로 신상품 업로드 알림
- 해시태그: 브랜드 태그 + 아이템 유형 + 일반 패션 태그
- 판매 완료 아이템은 "SOLD" 표기 (인기 증명 + FOMO 유발)
- 팔로워와의 소통 (댓글/DM)이 빈티지 비즈니스 핵심

## 5. 금지 패턴

브랜드 톤앤매너에 맞지 않는 카피 스타일:

| 금지 패턴 | 예시 | 대안 |
|---------|------|------|
| **컨디션 과장** | "새것 같은 상태!" (실제로 사용감 있음) | "자연스러운 빈티지감, 하자 없음" |
| **가격 중심 어필** | "구제라 이 가격! 개이득!" | "시간이 만든 가치를 합리적으로" |
| **무분별한 브랜드 표기** | 브랜드 확인 안 된 상품에 유명 브랜드 태그 | 확인된 브랜드만 표기, 미확인 시 "노브랜드 빈티지" |
| **공포 마케팅** | "이거 안 사면 평생 후회!" | "한 벌뿐인 아이템, 마음에 드시면 서두르세요" |
| **비하적 비교** | "패스트패션은 쓰레기" | "빈티지만의 특별함을 경험해보세요" |
| **연대/원산지 허위** | 확인 안 된 연대를 단정적으로 표기 | "90s 추정" 등 추정 표기 |
