## 1. 브랜드 미션

과거 시대의 아름다운 디자인 가구를 발굴하여 현대 공간에 새로운 가치를 부여한다. 대량생산 가구가 지배하는 시대에, 시간이 만들어낸 고유한 멋과 이야기를 가진 빈티지 가구를 큐레이션하여 고객의 공간을 세상에 하나뿐인 장소로 만든다.

**슬로건**: "과거의 멋을 현대로"

---

## 2. 톤앤매너

- **격식 수준**: 반격식체 (합니다체 기반이되 딱딱하지 않게)
- **브랜드 성격**: 감성적, 큐레이토리얼(curatrial), 전문적
  - 감성적: 가구가 가진 시대적 이야기와 공간의 분위기를 전달
  - 큐레이토리얼: 무작위 수집이 아닌, 안목 있는 선별
  - 전문적: 디자이너, 시대, 소재에 대한 지식을 자연스럽게 녹임
- **문장 예시** (권장 톤 참고용):
  1. "1960년대 덴마크에서 건너온 티크 사이드보드. 반세기가 지나도 변하지 않는 균형잡힌 비례감."
  2. "이 의자에는 수십 년의 시간이 만든 결이 있습니다. 새 가구에서는 느낄 수 없는 깊이."
  3. "매장에서 직접 만져보세요. 사진으로는 전할 수 없는 나무결과 질감이 있습니다."

## 3. 선호/금지 표현

### 선호 표현
| 표현 | 사용 맥락 |
|------|---------|
| "빈티지" | 기본 용어 -- "중고"가 아닌 "빈티지"로 통일 |
| "큐레이션" / "셀렉션" | 상품 선별 과정을 강조할 때 |
| "시간이 만든 멋" / "세월의 결" | 빈티지 특유의 사용감을 긍정적으로 표현 |
| "하나뿐인" / "유일한" | 일품 재고 특성 강조 |
| "공간에 이야기를 더하다" | 가구가 공간에 미치는 영향 |
| "미드센츄리 모던" / "북유럽 디자인" | 디자인 사조 언급 시 |
| "오리지널" / "정품" | 진품 여부 강조 |
| "컨디션" | 상태 설명 시 (한글 "상태"보다 전문적 느낌) |

### 금지 표현
| 표현 | 이유 |
|------|------|
| "중고" / "중고 가구" | 빈티지 가치를 훼손. 반드시 "빈티지"로 |
| "떨이" / "재고 처분" | 브랜드 가치 훼손 |
| "세일" / "할인" (과도한 사용) | 일품 재고는 가격 경쟁 불필요. 가치 기반 판매 |
| "저렴한" / "싼" | 가격 매력이 아닌 가치 매력을 강조 |
| "완벽한 상태" | 빈티지 특성상 보증 불가. "양호한 컨디션" 등으로 대체 |
| "새것 같은" | 빈티지의 가치는 새것이 아닌 데 있음 |
| "재입고 예정" | 일품 재고 특성상 불가능한 약속 |

## 4. 핵심 가치 제안

**메인 메시지**: "과거의 멋을 현대로" -- 수십 년의 시간을 견딘 디자인 가구가 당신의 공간에 유일무이한 이야기를 만듭니다.

**서브 메시지 3개**:
1. **큐레이션의 가치**: "수백 점 중 엄선한 한 점 -- 안목 있는 셀렉션으로 당신의 시간을 아껴드립니다"
2. **오프라인 체험**: "사진으로는 전할 수 없는 나무결, 질감, 스케일 -- 매장에서 직접 만나보세요 (11:00-18:00, 네이버 예약)"
3. **유일성**: "이 세상에 단 하나뿐인 가구 -- 지금 놓치면 다시 만날 수 없습니다"
