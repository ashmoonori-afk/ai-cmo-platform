# 카피 패턴 — 아이리스 코퍼레이션

## 헤드라인 패턴

### 문제 제기형 (B2B 장례식장 운영사 대상)
- "컨세션 운영에 인력과 비용을 쏟고 계십니까?"
- "지점마다 들쭉날쭉한 식음료 품질, 언제까지 감수하실 건가요?"
- "제사상 주문이 들어올 때마다 준비가 부담스러우신가요?"

### 결과 제시형
- "전국 9개 지점 운영 데이터로 검증된 메뉴 최적화"
- "위탁 후 컨세션 운영 비용을 줄이고 본업에 집중한 사례"
- "빈소 1건당 객단가 XX만원 달성한 메뉴 구성 전략"

### 신뢰/전문성 강조형
- "장례 현장만 바라본 전문 컨세션 운영사"
- "2023년부터 축적된 9개 지점 매출 데이터 기반 운영"
- "지역별·종교별 맞춤 메뉴 구성, 저희가 설계합니다"

### 파트너십 제안형
- "운영은 저희가, 수익은 함께"
- "위탁 운영 파트너십을 제안드립니다"
- "리스크를 나누고 안정적 수익을 만드는 구조"

## CTA 패턴
| 상황 | CTA 텍스트 |
|------|-----------|
| 위탁 운영 문의 | 운영 제안서 받아보기 / 무료 현장 진단 신청 |
| IR / 투자 문의 | 투자 자료 요청 / 미팅 일정 잡기 |
| 도입 사례 확인 | 운영 사례 살펴보기 / 지점 실적 데이터 보기 |
| 메뉴 컨설팅 | 메뉴 최적화 상담 신청 / 데이터 분석 결과 받기 |
| 제사상 문의 | 제사상 구성표 받기 / 제사상 패키지 안내 받기 |

## 이메일 제목 패턴 (B2B 아웃바운드)
- "[{장례식장명}] 컨세션 운영, 더 효율적인 방법이 있습니다"
- "전국 9개 지점 운영 데이터 기반 제안 — 아이리스 코퍼레이션"
- "빈소 객단가를 높이는 메뉴 구성 전략 공유드립니다"
- "[협업 제안] 장례 식음료 위탁 운영 파트너십 안내"
- "제사상 판매 개선 사례 — {지역명} 장례식장 적용 결과"

## 소셜 미디어 카피 구조

### LinkedIn (B2B 의사결정자 대상)
- Hook (1줄): 장례 현장 컨세션 운영의 핵심 문제 하나를 짚는다
- 본문 (3-5줄): 데이터 또는 현장 사례로 문제를 구체화하고 해결 방향 제시
- CTA: 자료 요청 또는 미팅 제안으로 마무리

**예시 구조:**
```
장례식장 컨세션 수익이 기대보다 낮은 이유는 메뉴 구성에 있습니다.

아이리스 코퍼레이션이 운영하는 9개 지점 데이터를 분석한 결과,
객단가 상위 지점과 하위 지점의 차이는 '메뉴 수'가 아니라
'카테고리 구성 비율'에서 나타났습니다.

구체적인 분석 내용을 공유드릴 수 있습니다.
관심 있으신 분은 메시지 남겨주세요.
```

### 업계 뉴스레터 / 협회 채널
- 장례 업계 운영 노하우 중심 콘텐츠
- 데이터 기반 인사이트 공유 (광고성 색채 최소화)
- 아이리스 코퍼레이션의 전문성을 간접적으로 드러내는 방식

## 제안서 / 피칭덱 카피 원칙
1. 첫 페이지에서 상대방의 문제를 먼저 짚는다 (공감 → 신뢰)
2. 데이터 없이 주장하지 않는다 ("지점 운영 실적 XX" 형식)
3. 위탁 파트너십의 구조를 명확히 — 역할 분담, 수익 배분 방식
4. 레퍼런스(기존 운영 지점 사례)를 필수로 포함
5. 다음 단계(Next Step)를 명확히 제시하며 마무리
