## 1. 네이버 SEO

- **알고리즘 3층**: C-Rank(출처=블로그 신뢰도: 주제 집중도·활동성·사용자 인기 — 공식 요인 The Egg 2020 정리) × D.I.A.(문서 선호도: 키워드 적합·경험 정보·충실성·독창성·적시성 — 트윈워드) + D.I.A.+(질의 의도 매칭, 2020.11~). 가중치·결합 방식 비공개 [추정]
- 유사문서 판독(2018.10~): 복제·중복은 원본만 노출
- **스마트블록/에어서치**: 2024.2부터 VIEW 전면 개편 — 공략법: 타깃 키워드를 실제 검색해 생성되는 블록 주제를 확인하고 그 세부 주제에 정확히 맞는 콘텐츠 제작 (어센트코리아)
- **2025.2 공식 발표: AI 자동 생성·실경험 없는 광고성 글 필터링 강화** (BIEN 2025 — 공식 블로그 2차 인용)
- 저품질: 공식 개념·기준 미공개. 실무 관측 트리거: 키워드 과반복 / 유사문서(복붙) / 상업 외부링크 남발 / 반응 품앗이 / 의료·금융 과장 표현 (TION STATION). 진단: 발행 글 제목 그대로 검색해 노출 확인
- **서치어드바이저 등록 절차**: searchadvisor.naver.com 소유확인(HTML/메타태그) → 사이트맵 제출(`/sitemap.xml`) → RSS 제출 → 개별 URL 수집 요청. 수집 ~14-16일 [2차], 수집≠노출 보장
- 네이버 vs 구글 점유율 최신 검증 수치: 모름

## 2. 구글 SEO (소규모 사이트)

- 2024.3 코어: Helpful Content가 코어에 통합(상시 평가), 저품질 45% 감소 집행 (Google 공식 + Search Engine Land 2024). E-E-A-T가 사실상 평가 프레임
- 테크니컬 최소 세트: GSC 등록 → sitemap → 색인 점검(URL 검사) → 모바일·HTTPS. 나머지는 콘텐츠 품질 후순위
- **2024 스팸 정책 3종** (공식 developers.google.com/search/docs/essentials/spam-policies):
  ① Scaled Content Abuse — "AI로 대량 발행" 전략 자체가 위반 소지 ② Site Reputation Abuse(기생 SEO) — 대형 매체 기고 순위 획득 금지, 2024.11 강화 ③ Expired Domain Abuse
- 코어 업데이트 피해 회복 통상 2-6개월 [2차]

## 3. GEO (AI 검색 인용 최적화)

- **GEO 논문 (arXiv:2311.09735, KDD 2024, 10,000 쿼리)** 가시성 개선:
  **인용문 추가 +41% / 통계 추가 +31% / 출처 인용 +30%** / 유창성 +28% / 쉬운 표현 +13% / **키워드 스터핑 −10%**
- **민주화 효과**: 검색 5위 사이트가 출처 인용 적용 시 +115.1%, 1위는 −30.3% — 후발·소상공인에 유리
- 도메인별 최적 기법: 사실형·법률=출처 / 토론=통계 / 인물·설명=인용문
- **llms.txt 판정: 효과 근거 없음** — 구글 Mueller "어떤 AI도 사용 안 함"(SEJ 2025); Ahrefs 실측 13.7만 도메인: 97%가 월간 요청 0건 (Ahrefs 2026) → 우선순위 최하위
- **AI 크롤러 robots.txt** (공식 문서):
  | 봇 | 역할 | 비고 |
  |----|------|------|
  | OAI-SearchBot | ChatGPT 검색 노출 | 차단 시 ChatGPT 검색 제외 (OpenAI 공식) |
  | GPTBot | GPT 학습 수집 | 검색 노출과 별개 |
  | PerplexityBot | Perplexity 검색 | robots.txt 준수 |
  | Perplexity-User | 사용자 실시간 요청 | "일반적으로 robots.txt 무시" (공식 명기) |
  | Google-Extended | Gemini 학습 제어 | **구글 검색 랭킹 무관 (공식)** |
  소상공인 기본값 = 전부 허용. 학습만 차단 시 GPTBot/Google-Extended만 Disallow
- **스키마의 AI 인용 효과: 근거 상충** — Ahrefs 1,885페이지 추적 "유의미 변화 없음"(2026) vs Google·MS 공식 "생성형 기능에 활용" → 판정: 위생 요소(Organization/FAQ/Article 3종만), 리소스는 본문 인용성에 투입 [추정]
- **네이버 AI 브리핑**: 2025.3.27 출시 → 2025.12 통합검색 쿼리 20% 돌파 (네이트뉴스) → 2026년 2배 목표. Cue:·클로바X는 2026.4 종료 통합 (ZDNet 2026.2). 인용 선정 기준 공식 미공개 — 모름 (C-Rank 신뢰 출처·목록/표 구조 유리 [추정 — 리드젠랩 2025.9, 스스로 추정 명시])
- 한국 실측 (Answer Global 2025.7, 주간 100회 조사): SEO 상위 달성 후 **Perplexity는 일관 인용(검색 순위 연동), ChatGPT 11%, Gemini 8%** — SEO≠GEO, 플랫폼별 전략 필요

## 4. AI 검색 트래픽 현실

- 절대 규모: 전체 웹 트래픽 ~1% (Search Engine Land 2025) / 성장: 미국 리테일 홀리데이 +693% YoY (Adobe 2026.1)
- **질 우위**: AI 검색발 방문자 전환 오가닉 대비 4.4배 (Semrush 2026); ChatGPT 유입 전환율 7.1% (Similarweb 2026)
- **제로클릭 심화**: AI 요약 노출 시 일반 결과 클릭 8%(미노출 15%) (Pew 2025.7, 6.9만 검색); AIO 존재 시 1위 CTR −58% (Ahrefs 2025.12); 구글 검색 중 외부 클릭 1/3 미만 (SparkToro 2026)
- 시사점: **"AI 답변 내 브랜드 언급/인용"을 별도 KPI로 승격** [추정 — 종합]

## 5. 콘텐츠 실전 공식

- 검색의도 4분류(정보/네비/상업조사/거래)별 포맷 매칭 — 불일치는 품질 무관 랭킹 불리 (Semrush, Backlinko)
- GEO 겸용 구조: **첫 단락에 직답(40-60단어) 선제시** → 근거 전개 (AI 발췌 단위)
- 제목 공식 (Backlinko 400만 검색결과): 40-60자 최고 CTR / 질문형 +14.1% / 괄호 +~40% (HubSpot [2차]) / 숫자 포함 우위
- **리프레시**: HubSpot 구 글 갱신으로 오가닉 +106%·리드 2배; 조회 76%가 기존 글에서; 콘텐츠 부패 6개월마다 클릭 20-30% 감소 [2차]; AI는 더 신선한 콘텐츠 인용 경향 [2차] → 규칙: 상위 20% 트래픽 글 분기별 갱신 [추정]
