# 메타 광고 실전 운영 리서치 — 2026-07-02

> 딥리서치 다이제스트 (검색 18회 + 원문 확인 6회). `playbooks/10-ads/meta-ads.md`의 근거 문서.
> Meta는 국가별 공식 단가·벤치마크를 공개하지 않음 — 한국 수치는 대행사 자체 추정 [추정].

## 1. 계정 세팅 순서 (초보가 막히는 지점)

1. 비즈니스 관리자(BM) 생성 → 페이지·인스타 연결 → 광고 계정 → 결제 등록 (PixelYourSite 2024-25)
2. 픽셀("데이터셋") 생성 — 이벤트 관리자 → 데이터 연결 → 웹 (Meta 공식 facebook.com/business/help/952192354843755)
3. 전환 API(CAPI): 2025년부터 이벤트 관리자에서 노코드 간편 연동 가능 (Grow with Sakib 2026, AdAmigo 2025)
4. 이벤트 검증: 테스트 이벤트로 픽셀+CAPI 쌍 "Deduplicated" 표시 확인 (FunnelFox 2025-26)
5. 도메인 인증: **2025-06 AEM 개편 후 웹 전환 필수 아님** — 링크 소유권/iOS 앱 용도만 (Adviso 2025)
6. 보안: 전원 2FA + **백업 관리자 1명** (개인 계정 제한 시 전체 잠김 방지 — Crystal Media 2025, 브런치 @powerblog/54)

## 2. 캠페인 구조 표준 (2024-2026)

- **Performance 5** (Meta Blueprint 공식): 계정 단순화 / 자동화(Advantage+) / 크리에이티브 다양화 / 데이터 품질(CAPI) / 성과 검증
- ASC는 2025년 "Advantage+ 판매 캠페인"으로 개편, 일반 생성 흐름에 통합 (Bir.ch 2026)
- 대원칙: **광고세트당 주 50회 전환**이 모이게 통합 — 쪼개면 학습 제한 (Tiger Pistol 2024; 한국어: 콘크리트파머스)
- 소액 실전 구조: 캠페인 1 + 세트 1-2 + 소재 5+ (브런치 @choi1z, 아이보스)
- **Andromeda 업데이트 (2025-10 전면 적용)**: "세트 1개 + 서로 다른 컨셉 소재 다수"가 표준. 세트 1개+소재 25개 = 전환 17%↑·비용 16%↓ (Meta 내부 테스트 2차 인용 — Jetfuel 2026)
- CBO vs ABO: 테스트는 ABO → 승자를 CBO로 스케일. 일 $100 미만이면 ABO (LeadEnforce 2025). CBO는 CPA 평균 4.6% 절감 (Madgicx 2025)
- ASC 필수: **기존 고객 예산 상한 20-30%** 미설정 시 리타겟에 예산 쏠림 (ATTN Agency 2025)

## 3. 타겟팅 (신호 축소 이후)

- Advantage+ 오디언스: 일예산 $50+ & 전환 데이터 있으면 우세; $30 미만·신규·니치는 상세 타겟 "제안 모드" 하이브리드 (Conversios 2026)
- 2025-03 상세 타겟 "제외" 기능 제거, 2025-06 관심사 카테고리 통폐합 (FanIQ 2025)
- 유사타겟: Andromeda 이후 1순위에서 밀림 (Alex Neiman 2026). 쓸 경우 시드 1,000-5,000명, 품질>크기
- 소액은 브로드 권장: 좁은 타겟 = 학습 제한 + 오디언스 중복 자기 경쟁 (Bestever 2025). 한국 실무 동일 결론 (브런치 @choi1z)

## 4. 예산·입찰 — 학습 단계 중심

- 학습 종료 = **7일 내 최적화 이벤트 50회** (Meta 공식 facebook.com/business/help/112167992830700)
- **최소 유효 일예산 = 목표 CPA × 50 ÷ 7** (Pigeon Digital 2026)
- 50전환 불가 소액 계정: ① 상위 퍼널 이벤트(장바구니 등)로 최적화 목표 하향 ② 세트 1-2개 극단 통합 ③ "학습 제한" 상태여도 CPA 목표 내면 운영 가능
- 한국 권장 구간 [추정 — SNS헬프 2026 자체 추정]: 첫 캠페인 월 30-50만(일 1-1.7만), 세트 1 + 소재 3종, 30일 → 확장. 실무자 일 3-10만 테스트 (브런치 @choi1z)
- **학습 리셋 주의**: 소재·타겟·이벤트 변경, 예산 ±20% 초과 변경 (Meta 공식 "영향이 큰 변경"; 한국어 해설 ampm). 구조 변경은 복제 후 수정

## 5. 크리에이티브

- **9:16 세로 영상+사운드**: 릴스 지면 이미지 대비 결과당 비용 34.5%↓ (Meta 공식 — facebook.com/business/ads/facebook-instagram-reels-ads)
- 세이프존 (2026-03 통합): 상단 ~14% / 하단 ~20-35% / 좌우 ~6% 텍스트·CTA 금지 (Billo 2026)
- 소재 수: 수동 세트당 3-6개, ASC 6-10개+; Andromeda 이후 "다른 컨셉 10-20개"가 레버 (Wonderful 2025)
- UGC형: 전환 우위 주장 [추정 — 벤더 데이터]; Meta 공식도 크리에이터 협업을 다양화 축으로 명시
- 카피: 기본 문구 모바일 ~125자 노출, 헤드라인 27-40자 (AdsUploader 2026). 한국어는 더 짧게 [추정]
- 교체 주기: 릴스 7-14일, 피드 이미지 14-28일 [추정 — AdLibrary 2026]; 실무 표준 빈도 기반 2-4주 로테이션

## 6. 측정 (iOS14+ 이후)

- **2025-06 AEM 대개편**: 8개 이벤트 제한·수동 우선순위 폐지 → 자동 집계. **"이벤트 우선순위 8개 세팅" 구버전 가이드는 이제 불필요** (Adviso, Success Local, Conversios 2025)
- 픽셀+CAPI 병행: 전환 포착률 95-99% 주장 [추정 — 벤더]; Meta 공식 권장
- **민감 업종 차단 (2025-01~)**: 헬스·웰니스 도메인은 구매·장바구니 등 하위 퍼널 이벤트 공유 차단 (픽셀·CAPI 모두) — 광고는 승인되는데 전환 데이터만 끊김 (Freshpaint, Polar Analytics 2025). 병원·건기식·피트니스 사전 진단 필수
- iOS 전환 최대 72시간 지연 → **최근 1-3일 성과로 판단 금지** (AppsFlyer)
- 벤치마크 (출처·기간 명시): 트래픽 CTR 1.71%/CPC $0.70, 리드 CTR 2.59%/CPL $27.66 (WordStream 미국 554캠페인 2025); 이커머스 CVR 중앙값 1.57%, ROAS 중앙값 ~1.93 (27five 2026); CPM 한국 $5.80 (Lebesgue 2026, 방법론 미공개); 한국 릴스 CPC ~980원 [추정 — SNS헬프 자체 추정]. **한국 공식 업종별 벤치마크: 모름 → 첫 30일 자체 데이터가 기준선**

## 7. 최적화 루틴

- 주간 점검 5종: 지출·CPA/ROAS(최근 3일 제외) / 학습 상태·주간 전환 수 / 빈도(2.5-3.0 소프트 가드, 3.0+&CTR 하락 = 소재 교체 — AdAmigo 2025) / 소재별 피로도 / 반려·결제 알림
- **끄기 룰**: 지출 ≥ 목표 CPA 2배 & 7일+ & CPA > 목표 1.5배 → 중단 (AdLibrary 2026)
- **증액 룰**: 5-7일 연속 목표 달성 세트만, 한 번에 10-20% (±20% 초과 = 학습 리셋), 48-72시간 관찰 (CausalFunnel 2026 + Meta 공식)

## 8. 정책·심사

- 심사 대부분 24시간 내 (Meta 공식)
- **반려 1위: 개인 특성(Personal Attributes)** — "당신의 관절염", "40대이신가요?" 류 (Meta 투명성 센터; AuditSocials 2026)
- 전후(Before/After) 사진·신체 클로즈업 반려 — 다이어트·미용 업종 최다 (ZapPush 2025-26)
- 재심사 가능(24-48h), **동일 광고 2차 어필 불가** (Get-Ryze 2026)
- 계정 보안 5칙: 전원 2FA / 백업 관리자 / 계정 공유 금지 / 지출 한도 / 권한 정기 감사 (2025년 광고계정 탈취 피싱 급증 — InfluencerMarketingHub)
