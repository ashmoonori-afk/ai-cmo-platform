# 네이버 검색광고(파워링크) 실전 리서치 — 2026-07-02

> 딥리서치 다이제스트 (검색 26회 + 원문 확인 14회). `playbooks/10-ads/naver-search-ads.md`의 근거 문서.
> 네이버 공식 도움말은 크롤러 차단 — 공식대행사 공지 전재·실무 커뮤니티(아이보스)·대행사 블로그 교차 검증. [미확인] = 공식 원문 미확인.

## 0. 기본 스펙

- 구조: 캠페인 → 광고그룹 → 키워드+소재(T&D), M:N 연결 (BAT batcrew.co.kr)
- CPC 과금(노출 무료), 입찰가 70원~10만원 (식스샵)
- 2가 경매: 실 과금은 차순위 입찰가·품질지수에 따라 입찰가보다 낮게 형성 (티온 2026). "+10원" 공식 문구 [미확인]
- 노출: PC 통합검색 최대 10개(저검색량 3개), 모바일 최대 5개 (아이보스). 순위 = 입찰가 × 품질지수

## 1. 계정 구조

- 캠페인: "파워링크_PC / 파워링크_MO" 2개로 크게 시작 → 그룹에서 세분화 (BAT, 브런치 2023)
- **그룹 = "동일 소재를 보여줄 키워드 묶음"** (그룹 분리의 유일 원칙). 그룹당 키워드 최대 1,000개 (식스샵)
- **키워드는 그룹 간 이동 불가, 삭제 후 재등록 시 품질지수·성과 초기화** → 초기 설계가 결정적 (식스샵)
- 키워드 3분류: 브랜드(저CPC·고전환) / 대표(고경쟁·고CPC) / 세부(저경쟁 — "촘촘한 그물") (브런치 2025-06)
- 경쟁사 브랜드 키워드: 대부분 업종 등록 제한 + 상표권 판례 리스크 (nepla); 역으로 침해당하면 '스크럽' 신고 (오픈애즈)
- 그룹당 적정 키워드 수 공인치: 모름

## 2. 키워드 실전

- 공식: **대표 키워드 순위 3~6위 하향 + 세부 키워드 다수 등록** (커넥트리 2026-04, 가비아). 100만원 시뮬: 핵심만 → 전환 12 vs 병행 → 전환 30 [대행사 예시 [추정]]
- 키워드도구([도구>키워드도구]) + 자동완성·데이터랩·아이템스카우트 병용
- **확장검색 (2024-10 도입 — 최대 변화)**: 랜딩페이지 내용 기반 AI 자동 노출, 자동입찰(그룹 예산 비율로 간접 통제), '일치 검색' 설정 시 비활성 (밀크애드 공지 전재)
- **제외 검색어**: [광고그룹 > 제외 검색어 탭]. 주 1~2회 검색어 리포트 확인 → "무료/후기/뜻" 등 구매의도 낮은 검색어 차단이 CPC 방어 핵심 (고도몰 2025, 티온 2026)

## 3. 입찰·예산

- 시스템 참고치 3종: 최소노출입찰가 / 중간입찰가 / 순위별 평균 (노출 비보장 — ampm)
- 실전: **70원 시작 → 하루 100원씩 증액 7일 테스트**; "PC 10위·모바일 3위 입찰가" 저비용 유지 전략 (아이보스)
- CPC 현실 (2026, 대행사 집계 [추정]): 평균 — 법률 ~3,800원 / 인테리어 ~2,200원 / 이사·청소 ~1,300원 / 음식점 ~700원. 핵심 키워드 상위노출 — 법률 5-10만원 / 인테리어 3-6만원 / 병원 1-4만원 (티온, 커넥트리). 전체 평균 ~530원, CTR 0.8-2.4% (ampm)
- 하루예산 + **균등배분** 옵션 = 소액 필수 (조기 소진 방지). 월 30만원 ≈ 566클릭 [추정]
- 요일/시간대 입찰가중치로 전환 없는 시간대 OFF (아이보스)

## 4. 광고문안(T&D) + 확장소재

- **제목 ≤15자, 설명 20~45자** (매드타임스, 아이보스)
- 공식: 검색 키워드를 제목·설명에 포함 / 구체 혜택 / 차별점 1개 / 행동 동사
- **키워드 삽입**: `[키워드(대체키워드)]` — 검색어 자동 삽입, 초과 시 대체어 (ampm)
- T&D의 약속 = 랜딩페이지에 그대로 있어야 함 (없으면 클릭만 나가고 전환 0)
- 확장소재 10종 (2025): 서브링크(4)·추가제목·홍보문구·이미지·추가설명·쇼핑정보·전화번호·이미지형 서브링크·플레이스·블로그 리뷰 + 가격링크(3) (퍼즐 2025). 전화 유입 업종 = 전화+위치 우선, 쇼핑몰 = 서브링크+가격링크 [추정]
- 심사 반려: 의학적 효능 / 근거 없는 최상급("1위" — 출처 필요) / 공포 조장 / 저품질·합성 이미지·이미지 내 전화번호 / 민감 업종(건기식·의료·부동산·금융) (ampm 반려 사례집)

## 5. 품질지수

- 7단계, 신규 4칸 시작, 24시간 내 측정 (오픈애즈, 새벽네시 2024)
- 효과: 동일 입찰가에 상위 + **실 CPC 하락** ("100원×3점=50원×6점" 설명 모델 [추정], 공식 산식 비공개)
- 구성: CTR / 키워드↔소재 관련성 / 키워드↔랜딩 관련성
- 올리기: ① 키워드 단어를 소재에 반복 ② 키워드별 랜딩을 최종 상세로 정확 연결 ③ 저CTR 소재 A/B 교체 ④ 랜딩 속도·모바일

## 6. 전환추적 + 루틴

- **프리미엄 로그분석 (무료)**: 비즈채널 등록·검수 → 신청 → 스크립트 설치(아임웹/카페24 등 임대몰은 인증키만) → 검수 (아임웹 FAQ, naver.github.io 공식 가이드)
- 직접전환 = 클릭 후 30분 내 / 간접전환 = 이후~기여기간(기본 15일, 7-20 조정) [미확인 — 단일 소스]
- 일일 5-10분: 어제 지출 확인 / 비효율 순위 하향 / 우수 상향 / 미노출 기회 / 키워드 발굴 (브런치 2023)
- 주간: 검색어 리포트 → 제외 처리; 다차원 보고서 시간대별 → OFF; **켜고 끄기는 CTR이 아니라 CVR 기준, 2-4주 데이터 후** (브런치 2025-10)
- [도구 > 자동 규칙]: 조건 시 알림/입찰 변경/OFF/예산 변경 (아이보스)

## 7. 실수 + 사기 방어

- 실수 Top: 대표 키워드 1-2위 고집 / 전환 설계 없이 클릭만 / 전 키워드 한 그룹 / 부분 설정 후 방치 / 검색어 리포트 미확인(확장검색 이후 치명적)
- **공식대행사는 광고주에게 수수료를 받지 않음** (네이버가 소진액 15% 지급 — 커넥트리, 아이보스) → 수수료 요구·연납 유도("연 200만원 올인원")·선결제 장기계약 = 경고 신호 (온엠디 2023)
- 계정 소유권(아이디)은 반드시 광고주 본인 보유. 무효클릭은 클린센터 조사 요청 + IP 차단 기능

## 8. 2024-2026 타임라인

- 2024-07: 임대몰 전환 이벤트 6종 확대 (naver.github.io)
- **2024-10: 확장검색 도입** — 제외 검색어 관리 루틴 필수화
- 2024-11: 검색광고 ID 네이버 통합회원 전환
- 2025-02: 메시지 광고 관리 오픈
- **2026-03-30: 광고주센터 통합 개편** (검색+디스플레이 단일 UI, 성과 대시보드, 자동화 인사이트 — ampm 2026)
- 쇼핑몰 병행 정석: 구매형 검색어 = 쇼핑검색광고 / 정보형·브랜드 = 파워링크 (ampm)
