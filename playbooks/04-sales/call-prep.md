# 콜 프렙 플레이북

## 목적

미팅 전 1페이지 브리프를 생성하여 세일즈 담당자가 회의에 완벽히 준비된 상태로 임할 수 있도록 한다.
최종 산출물: 기업 개요, 토킹 포인트, 예상 Q&A, 다음 스텝 시나리오를 포함한 1페이지 미팅 브리프

## 에이전트 조합

```
researcher → sales-writer
```

| 에이전트 | 역할 | 입력 | 출력 |
|---------|------|------|------|
| researcher | 타겟 기업 및 담당자 리서치 (INT-03 연동) | 회사명, 담당자명, 미팅 목적 | 기업 리서치 데이터 (INT-03 JSON) |
| sales-writer | 브리프 문서 작성 | INT-03 출력 + 미팅 컨텍스트 | 1페이지 미팅 브리프 |

## 입력

- 타겟 회사명, 담당자명, 직함
- 미팅 일시 및 목적 (초회 미팅 / 팔로업 / 제안서 발표 등)
- 이전 대화 이력 (있는 경우)
- `intelligence/account-research/{날짜}_{회사명}_INT-03.json` (존재 시 활용, 없으면 researcher가 즉시 수집)

## 참조 문서

- `clients/{client}/config.md` — pain points 및 이상적 고객 기준
- `clients/{client}/config.md` — 경쟁사 대비 차별점 (예상 질문 대비)

## 프레임워크

```
VST SAL-02 패턴 기반 1페이지 미팅 브리프:

섹션 1 — 기업 개요 (5줄)
  1. 회사명, 산업, 규모 (직원 수 또는 매출)
  2. 주요 제품/서비스 1~2개
  3. 최근 주목할 뉴스 또는 이벤트
  4. 성장 단계 및 현재 도전 과제
  5. 미팅 담당자 역할 및 결정권 수준

섹션 2 — 미팅 목적
  - 이번 미팅에서 달성해야 할 단일 목표 (예: 다음 단계 합의, 니즈 확인, 예산 파악)
  - 성공 기준: "이 미팅이 성공이면 {결과}를 얻는다"

섹션 3 — 토킹 포인트 3개
  - TP1: [pain point 연결] 질문 또는 인사이트 공유
  - TP2: [솔루션 가치] 구체적 사례 또는 수치 제시
  - TP3: [다음 스텝 유도] 합의 또는 후속 액션 제안

섹션 4 — 예상 질문 + 답변
  - Q1: [가장 빈번한 반론] → A: [간결한 답변 + 증거]
  - Q2: [가격/비용 관련 질문] → A: [ROI 프레임으로 전환]
  - Q3: [경쟁사 비교 질문] → A: [차별점 2가지]
  - Q4: [도입 리스크/기간 질문] → A: [단계적 접근 제안]

섹션 5 — 다음 스텝 시나리오
  - 시나리오 A (긍정적): {조건} → {액션} (예: 제안서 발송, 파일럿 제안)
  - 시나리오 B (중립적): {조건} → {액션} (예: 추가 정보 공유, 재미팅 일정)
  - 시나리오 C (부정적): {조건} → {액션} (예: 3개월 후 재접촉, nurture 시퀀스로 이동)
```

## 출력 템플릿

```markdown

## 기업 개요

| 항목 | 내용 |
|------|------|
| 회사명 | {회사명} |
| 산업/규모 | {산업} / {직원 수 또는 매출} |
| 주요 제품 | {제품/서비스 1~2개} |
| 최근 이슈 | {최근 뉴스 또는 이벤트} |
| 도전 과제 | {현재 직면한 주요 문제} |
| 담당자 | {이름} — {직함} (결정권: {높음/중간/낮음}) |

## 미팅 목적

**단일 목표:** {이번 미팅에서 반드시 달성할 것}

**성공 기준:** 이 미팅이 성공이면 _{결과}_ 를 얻는다.

---

## 토킹 포인트

**TP1 — Pain Point 연결**
"{질문 또는 인사이트}" → 예상 반응: {긍정/중립/방어}

**TP2 — 솔루션 가치**
"{사례 또는 수치}" → 핵심 메시지: {1문장}

**TP3 — 다음 스텝 유도**
"{합의 또는 액션 제안}" → 목적: {다음 단계로 이동}

---

## 예상 Q&A

**Q1: {가장 빈번한 반론}**
A: {간결한 답변} — 증거: {수치 또는 사례}

**Q2: {가격/비용 관련}**
A: {ROI 프레임} — 예: "월 {비용} 투자 대비 {효과}"

**Q3: {경쟁사 비교}**
A: {차별점 1} + {차별점 2}

**Q4: {도입 리스크 또는 기간}**
A: {단계적 접근 또는 파일럿 제안}

## 다음 스텝 시나리오

| B — 중립적 | {추가 검토 필요 / 내부 공유 필요} | {자료 발송 / 재미팅 일정 잡기} |
```
