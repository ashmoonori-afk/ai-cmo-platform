# 미팅 후 처리 플레이북

## 목적

미팅 메모(또는 트랜스크립트)를 입력받아 3가지 산출물을 동시에 생성한다.
하나의 파일에 CRM 요약, 팔로업 이메일, 내부 공유 메모를 포함하여 미팅 후 처리를 완전히 자동화한다.
최종 산출물: CRM 요약 (5줄) + 팔로업 이메일 + 내부 공유 메모

## 에이전트 조합

```
sales-writer → reviewer
```

| 에이전트 | 역할 | 입력 | 출력 |
|---------|------|------|------|
| sales-writer | 미팅 메모 분석 후 3가지 산출물 동시 작성 | 미팅 메모 / 트랜스크립트 | CRM 요약 + 팔로업 이메일 + 내부 메모 초안 |
| reviewer | 품질 검증, 톤 확인, 팩트 확인 | 초안 3종 | 최종 산출물 3종 |

## 입력

- 미팅 메모 또는 통화 트랜스크립트 (자유 형식 텍스트)
- 참석자 정보 (우리 측 + 고객 측)
- 미팅 일시
- 현재 딜 스테이지 (CRM 기준)

## 참조 문서

- `clients/{client}/brand-guidelines.md` — 이메일 톤앤매너 및 서명
- `clients/{client}/brand-guidelines.md` — 팔로업 이메일 구조

## 프레임워크

```
VST SAL-03 패턴 기반 3가지 산출물 동시 생성:

[입력] 미팅 메모 / 트랜스크립트
        ↓
[sales-writer] 분석 및 3가지 산출물 작성
        ↓
산출물 1: CRM 요약 (5줄)
  - 라인 1: 참석자 (우리 측 + 고객 측)
  - 라인 2: 핵심 논의 내용 (1~2문장)
  - 라인 3: 다음 스텝 + 담당자 + 마감일
  - 라인 4: 딜 상태 (스테이지 변경 여부 + 딜 규모 추정)
  - 라인 5: 성사 확률 및 주요 리스크/블로커

산출물 2: 팔로업 이메일
  - 발송 타이밍: 미팅 후 24시간 이내
  - 제목: "Re: [미팅 주제] — 말씀드린 다음 단계 공유"
  - 구조: 감사 인사 → 핵심 합의 요약 → 다음 스텝 명시 → CTA
  - 톤: professional-warm (brand-guidelines.md 기준)
  - 길이: 200~300자

산출물 3: 내부 공유 메모
  - 핵심 인사이트 (경쟁사 언급, 예산 힌트, 의사결정 구조 등)
  - 액션 아이템 (담당자 + 마감일)
  - 팀 공유 포인트 (마케팅/제품 팀에 전달할 고객 피드백)
  - 리스크 플래그 (주의가 필요한 신호)
```
