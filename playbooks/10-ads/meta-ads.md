# 메타 광고(페이스북·인스타그램) 운영 플레이북

## 목적

메타 광고 계정 세팅부터 캠페인 구조·소재·주간 운영까지 **사장님 대신** 준비한다. 산출물은 광고 관리자에 그대로 입력 가능한 완성 상태(`prompts/shared/deliverable-standard.md`).

> 근거: `docs/research/2026-07-02-meta-ads-research.md` (2026-07 딥리서치 — Meta 공식 문서·Blueprint·글로벌 벤치마크. 한국 단가는 대행사 추정 [추정])

## 에이전트 조합

```
strategist → copywriter → reviewer
```

| 순서 | 에이전트 | 역할 | 모델 |
|------|---------|------|------|
| 1 | strategist | 예산 구간별 캠페인 구조·타겟 설계 | sonnet |
| 2 | copywriter | 소재 브리프 6컨셉 + 카피 | sonnet |
| 3 | reviewer | 정책 사전 심사 + 실행성 검증 | sonnet |

## 프레임워크

### Step 1: 계정 세팅 체크리스트

순서 고정: 비즈니스 관리자(BM) 생성 → 페이지·인스타 연결 → 광고 계정·결제 → 픽셀(데이터셋) 생성 → **전환 API(CAPI) 노코드 연동** → 테스트 이벤트로 "Deduplicated" 확인 → 전원 2FA + **백업 관리자 1명**.

- 도메인 인증은 2025-06 AEM 개편 후 웹 전환 필수 아님 (링크 소유권·iOS 앱 용도만)
- **구버전 가이드 경고**: "이벤트 우선순위 8개 세팅"은 폐지된 절차 — 하지 않는다
- **민감 업종 사전 진단** (`industry_sensitive`): 헬스·웰니스 도메인은 2025-01부터 구매·장바구니 등 하위 퍼널 이벤트 공유 차단 — 광고는 승인되는데 전환 데이터만 끊김. 해당 시 리드 폼·전화 전환 등 대체 설계

### Step 2: 캠페인·세트·광고 갯수 전략 (strategist)

| 월 예산 | 구조 | 최적화 이벤트 |
|--------|------|-------------|
| 30~50만 원 | 캠페인 1 + 세트 1 + 소재 3~5 (ABO) | 상위 퍼널 (장바구니·문의 시작 — 주 50회 달성 가능한 것) |
| 50~150만 원 | 캠페인 1 + 세트 1~2 (ABO 테스트) + 소재 6~10 | 목표 전환 또는 상위 퍼널 |
| 150만 원+ | Advantage+ 판매 캠페인 + **기존 고객 상한 20~30%** + 승자 CBO 스케일 | 목표 전환 |

**갯수 결정 원칙 (많이 만들지 말 것 — 세트는 줄이고 소재는 다양하게):**
- 세트 수 = "주 50회 전환이 모이는 최소 개수" (학습 단계 공식 — Meta 공식). 최소 유효 일예산 = 목표 CPA × 50 ÷ 7
- 소재 수: 수동 세트당 3~6개 / ASC 6~10개+ — Andromeda(2025-10) 이후 "비슷한 변형 N개"가 아니라 **서로 다른 컨셉**이 레버 (세트 1 + 소재 25개 = 전환 17%↑·비용 16%↓, Meta 내부 테스트 2차 인용)
- 캠페인 수: 목표(전환/리드/트래픽)당 1개가 기본. 같은 목표로 캠페인을 복수 운영하면 오디언스 중복 자기 경쟁 [추정 — 중복 경쟁 근거는 세트 단위로 확인됨]
- 테스트 규칙: 한 번에 바꾸는 변수 1개 (소재 또는 타겟), 판정은 최소 7일 + 최근 3일 제외
- 50전환 불가 소액 계정: "학습 제한" 상태여도 CPA가 목표 안이면 운영 가능 — 실패 판정 아님

### Step 2.5: KPI 설정 (캠페인 시작 전 확정)

**KPI 트리 — 최종 지표에서 역산:**

```
최종 KPI:   CPA(전환당 비용) 또는 ROAS      ← 사업 목표
  ↑ 중간:   CVR(전환율) × CPC(클릭당 비용)   ← CPA = CPC ÷ CVR
  ↑ 상위:   CTR(클릭률) × CPM(1천 노출 비용) ← CPC = CPM ÷ (CTR × 10)
```

- **목표 CPA 산정**: 객단가 × 마진율 × 허용 비중 (예: 5만 원 × 40% 마진 × 50% 허용 = 목표 CPA 1만 원) [산술 공식]
- **목표 ROAS 하한**: 이커머스 콜드 타겟 2X면 양호, 웜 3X+ (27five 2026 집계)
- 글로벌 참고치: 트래픽 CTR ~1.7%·리드 CVR ~7.7% (WordStream 2025) / 이커머스 CVR 중앙값 1.57% (27five 2026)
- **한국 공식 벤치마크 없음 → 첫 30일 자체 데이터가 기준선.** KPI 표에 "목표 / 30일 실측 / 차이" 3칸을 만들어 납품

### Step 2.7: 소재별 지표 읽는 법 (진단표 — 산출물에 포함)

소재별로 매주 이 칼럼을 본다: 지출 / 노출 / CPM / CTR(링크) / CPC / 빈도 / 전환 / CPA / (영상) 3초 조회율·평균 시청 시간.

| 증상 (지표 조합) | 진단 | 처방 |
|----------------|------|------|
| CTR 낮음 + CPM 정상 | 소재가 시선을 못 잡음 | 첫 3초·썸네일·훅 문구 교체 (다른 컨셉으로) |
| 3초 조회율 낮음 (영상) | 도입부 이탈 | 결과를 먼저 보여주는 구조로 재편집 [실무 관행] |
| CTR 좋음 + CVR 낮음 | 소재-랜딩 불일치 | 랜딩 첫 화면에 소재의 약속을 그대로 배치 |
| CPM 급등 | 오디언스 피로 또는 경매 경쟁 심화 | 타겟 확장(브로드) 또는 소재 전면 리프레시 |
| 빈도 3.0+ & CTR 하락 | 소재 피로 | 해당 소재 교체 (AdAmigo 2025 벤치마크: 프로스펙팅 2.5~3.0 가드) |
| 전환 있는데 "학습 제한" | 데이터 볼륨 부족일 뿐 | CPA 목표 안이면 유지, 아니면 이벤트 상향 통합 |
| 최근 1~3일 성과 급락처럼 보임 | iOS 지연 집계 (최대 72시간) | 판단 보류 — 3일 지나고 재확인 |

판정 우선순위: **CPA/ROAS(최종) > CVR > CTR** — CTR 높은 소재가 아니라 CPA 낮은 소재가 승자다.

### Step 3: 타겟 세팅 시트

- 분기: 일예산 ~7만 원+ & 전환 데이터 있음 → Advantage+ 오디언스 / 일 4만 원 미만·신규·니치 → 상세 타겟 "제안 모드" 하이브리드
- 소액은 브로드 기본 (좁은 타겟 = 학습 제한 + 오디언스 중복 자기 경쟁)
- 유사타겟은 1순위 아님. 쓸 경우 시드 1,000~5,000명(구매자 우선)
- 2025년 폐지 기능 주의 표기: 상세 타겟 "제외", 관심사 통폐합

### Step 4: 소재 브리프 6컨셉 + 카피 (copywriter)

서로 다른 6컨셉: UGC 후기형 / 문제-해결형 / 데모형 / 오퍼형 / 비교형 / 브랜드형

- 규격: **9:16 세로 영상 우선** (릴스 지면 이미지 대비 결과당 비용 34.5%↓ — Meta 공식), 세이프존(상단 ~14%·하단 ~20-35%·좌우 ~6% 텍스트 금지)
- 카피: 기본 문구 첫 125자 안에 훅 (한국어는 더 짧게 [추정]) / 헤드라인 27~40자
- 첫 3초 스크립트 포함, 후킹문안 리스트의 Top 문안 재사용
- 교체 주기 가이드: 빈도 3.0 초과 + CTR 하락 시 교체, 2~4주 로테이션

### Step 5: reviewer 정책 사전 심사

- **개인 특성(Personal Attributes) 감지**: "당신의 ○○(질환·나이·체형·재정)" 류 — 반려 1위. 한국어 예시 목록 대조
- 전후(Before/After) 사진·신체 클로즈업 금지 (다이어트·미용 최다 반려)
- 재심사 절차 안내 (24-48시간, **동일 광고 2차 어필 불가**)
- deliverable-standard 3.5단계 + "매출 보장" 문구 금지

## 운영 루틴 (산출물에 포함)

**주간 점검 5종** (iOS 전환 최대 72시간 지연 — **최근 3일 데이터로 판단 금지**):
1. 지출·CPA/ROAS vs 목표 (최근 7일, 최근 3일 제외)
2. 학습 상태 + 세트별 주간 전환 수
3. 빈도 (2.5~3.0 소프트 가드)
4. 소재별 성과 → 피로 소재 교체 예약
5. 반려·결제 알림

**끄기 룰**: 지출 ≥ 목표 CPA 2배 & 7일+ & CPA > 목표 1.5배 → 중단
**증액 룰**: 5~7일 연속 목표 달성 세트만, 한 번에 10~20% (**±20% 초과 변경 = 학습 리셋**), 48~72시간 관찰
**학습 리셋 경고**: 학습 중 소재·타겟·이벤트·예산(±20%+) 수정 금지. 구조 변경은 복제 후 수정.

## 성과 기준선

- 글로벌 참고치 (출처: WordStream 2025 미국, 27five 2026): 트래픽 CTR ~1.7% / 리드 CVR ~7.7% / 이커머스 ROAS 중앙값 ~1.9 (콜드 2X 양호)
- **한국 공식 업종별 벤치마크는 없음** — 첫 30일 자체 데이터가 기준선. 30일 후 비교 리포트 포맷 동봉

## 산출물 (묶음)

```
1. 사장님용 한 장 요약
2. 계정 세팅 체크리스트 (민감 업종 진단 포함)
3. 캠페인·세트·광고 갯수 구조안 (예산 구간별) + 타겟 세팅 시트
4. KPI 설정표 (목표 CPA 역산 + 목표/30일 실측/차이 3칸)
5. 소재 브리프 6컨셉 + 카피 (세이프존 규격)
6. 소재별 지표 진단표 (증상→진단→처방, 주간 판독용)
7. 주간 점검 체크리스트 (끄기·증액 룰 포함)
8. 정책 사전 심사 체크리스트 + 계정 보안 5칙 (2FA·백업 관리자·공유 금지·지출 한도·권한 감사)
9. 성과 기준선 문서 (30일 후 비교 포맷)
```
