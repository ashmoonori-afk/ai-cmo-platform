# 네이버 검색광고(파워링크) 운영 플레이북

## 목적

네이버 검색광고를 **사장님 대신** 설계·세팅·운영한다. 산출물은 광고시스템에 그대로 입력 가능한 완성 상태(`prompts/shared/deliverable-standard.md`)이며, 주간 10분 점검 루틴까지 포함한다.

> 근거: `docs/research/2026-07-02-naver-ads-research.md` (2026-07 딥리서치 — 공식대행사 공지·아이보스·대행사 소스 교차 검증. CPC 수치는 대행사 집계 [추정])

## 에이전트 조합

```
seo-specialist → copywriter → reviewer
```

| 순서 | 에이전트 | 역할 | 모델 |
|------|---------|------|------|
| 1 | seo-specialist | 키워드 시트(3분류)·입찰 설계·구조도 | sonnet |
| 2 | copywriter | T&D 문안·확장소재 문안 | sonnet |
| 3 | reviewer | 심사 반려 사전 검사 + 실행성 검증 | sonnet |

## 프레임워크

### Step 1: 캠페인 구조도 (seo-specialist)

- 캠페인은 "파워링크_PC / 파워링크_MO" 2개로 크게 시작, 광고그룹에서 세분화
- **그룹 = 동일 소재를 보여줄 키워드 묶음** (유일한 분리 원칙). 그룹별 랜딩 URL·기본 입찰가·소재노출방식 명기
- **경고 내장**: 키워드는 그룹 간 이동 불가, 삭제 후 재등록 시 품질지수 초기화 — 처음 설계가 결정적

### Step 2: 키워드 시트 30~50개 (3분류)

- 브랜드 / 대표 / 세부 3분류 × (월간검색수, 예상 CPC 구간 [추정 표기], 초기 입찰가, 목표 순위, 랜딩 URL)
- **대표 키워드 목표 순위는 3~6위 기본** (1~2위 고집 = 초보 적자 1위 원인)
- 초기 입찰가: 70원 시작 → 하루 100원씩 증액 7일 테스트 플랜
- 경쟁사 브랜드명 등록 금지 (정책 제한 + 상표권 판례 리스크). 역침해 발견 시 '스크럽' 신고 안내

### Step 3: 제외 검색어 시드 리스트 (확장검색 대응)

2024-10 확장검색 도입 이후 필수: 업종별 구매의도 낮은 검색어 20개+ ("무료", "후기", "뜻", "채용", "직영 아닌 지역명" 등). [광고그룹 > 제외 검색어 탭] 등록 스텝 포함.

### Step 4: T&D 문안 5세트 (copywriter)

- 규격: **제목 ≤15자 / 설명 20~45자**, 그룹별 A/B 2안
- 공식: 키워드를 제목·설명에 포함 + 구체 혜택 + 차별점 1개 + 행동 동사
- 다품목이면 키워드 삽입 구문 `[키워드(대체키워드)]` 적용
- **T&D가 약속한 내용이 랜딩에 실제로 있는지 대조표** 첨부 (불일치 = 클릭비만 나감)

### Step 5: 확장소재 플랜

- 전화 유입 업종(음식점·수리·상담): 전화번호 + 위치정보 우선
- 쇼핑몰: 서브링크(4) + 가격링크(3) 우선
- 각 소재 입력값 초안 완성 상태로 납품

### Step 6: 입찰가·일예산 계산표

- 월 예산 → 일예산 환산 + **균등배분 ON** (소액 필수 — 조기 소진 방지)
- 요일/시간대 가중치 초안 (업종 영업시간 기반, 전환 데이터 축적 후 조정)

### Step 7: 전환추적 설치 체크리스트

- 비즈채널 등록·검수 → [도구 > 프리미엄 로그 분석] 신청(무료) → 스크립트/인증키 설치(아임웹·카페24 등 임대몰 분기) → 검수 요청
- **전환추적 없이 광고 시작 금지** (초보 실수 2위)

### Step 8: reviewer 심사 사전 검사

- 반려 사유 체크: 의학적 효능 표현 / 근거 없는 최상급("1위" — 출처 명기 필요) / 공포 조장 / 이미지 내 전화번호·URL / 민감 업종(건기식·의료·부동산·금융) 표현
- deliverable-standard 3.5단계 (즉시 입력 가능 상태·"매출 보장" 문구 금지)

## 운영 루틴 (산출물에 포함)

- **주간 10분**: 검색어 리포트 → 무관 검색어 제외 / 시간대별 보고서 → 전환 없는 시간대 OFF / 품질지수 3칸 이하 그룹 리스트업
- **켜고 끄기 판정: CTR이 아니라 CVR 기준, 2~4주 데이터 축적 후**
- **월간**: 키워드 전수 판정(확대/유지/축소/OFF) + 소재 A/B 승자 교체 + 리포트 (정량+쉬운 해석+다음 달 액션 3개)
- [도구 > 자동 규칙]로 알림·자동 OFF 설정 안내

## 사기 방어 1페이지 (필수 동봉)

- **공식대행사는 광고주에게 수수료를 받지 않는다** (네이버가 소진액의 15%를 대행사에 지급) → 수수료·연납("연 200만원 올인원")·선결제 장기계약 요구 = 경고 신호
- 광고 계정 아이디는 반드시 사장님 본인 소유
- 무효클릭 의심 시 클린센터 조사 요청 + IP 차단 기능
- `playbooks/09-local/support-programs.md`의 감별표와 연동

## 산출물 (묶음)

```
1. 사장님용 한 장 요약 (내 광고 구조, 월 예산, 이번 주 할 일 하나)
2. 캠페인 구조도 + 키워드 시트 30~50개 + 제외 검색어 시드
3. T&D 5세트 (A/B) + 확장소재 입력값
4. 입찰·일예산 계산표 + 전환추적 설치 체크리스트
5. 주간 10분 점검 체크리스트 + 월간 리포트 템플릿
6. 사기 방어 1페이지
```
