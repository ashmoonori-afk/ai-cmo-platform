# 당근 비즈프로필·동네 광고 플레이북

## 목적

동네 반경 고객에게 닿는 당근 비즈프로필을 **사장님 대신** 세팅하고, 소식·쿠폰·광고 소재까지 만들어 납품한다. 하이퍼로컬(동네 단위) 타겟이 구조적으로 내장된 유일한 대형 플랫폼.

근거: 당근 MAU 2,031만 (전자신문 2025-11, etnews.com/20251112000220), 비즈프로필 누적 약 265만 개·2025년 광고주 수 +37% (벤처스퀘어, venturesquare.net/1069052). 공식 표기 "전국 비즈프로필 170만+, 단골 980만+" (business.daangn.com/business-profile/about — 집계 기준 차이 [추정]).

## 프레임워크

### Step 1: 비즈프로필 세팅 시트

절차: 당근 앱 → "나의 당근" → 사장님 메뉴 → 비즈프로필 만들기 → 이름·활동 지역·카테고리 → 가게 사진 (실무: mimmi.co.kr/당근-비즈니스-비즈프로필-등록-방법/).
등록·기본 사용 무료, 앱 내 상품 판매 시 수수료 3.3% (실무: loadofmoney.com 가이드) — 비용 투명성 표기.

산출물에 미리 채워 주는 값: 프로필 이름 / 활동 지역 / 카테고리 / 소개 200자 / 사진 촬영 목록 5장 (외관·대표상품·내부·사장님·가격표).

### Step 2: 소식 4주분 (동네 말투)

- 주 1회 × 4건 — 당근 특유의 이웃 말투 (격식체보다 친근체, 과장 금지)
- 비즈프로필만 만들고 소식 0건 방치가 흔한 실수 [추정 — 광고 클릭이 와도 전환할 콘텐츠가 없음]
- 소식·쿠폰·단골 기능 묶음이 플랫폼 공식 재방문 루프 (business.daangn.com/business-profile/about)

### Step 3: 쿠폰 2종 + 채팅 스크립트

- 첫 방문 쿠폰 + 단골 전환 쿠폰 문구 (혜택·기간·조건 명시)
- 채팅 응대 스크립트 5종: 영업시간 문의 / 가격 문의 / 예약 / 컴플레인 / 부재중

### Step 4: 동네 광고 세팅안

- 타겟 동네 리스트 (상권 반경에 정확히 좁히는 것이 효율 핵심 — 아이보스 i-boss.co.kr/ab-2987-377223)
- 모드 선택 안내: 간편모드는 CPC 단가 조정이 유연하지 않아 예산 통제 어려움 → **전문가모드 + 최소 일예산 1만 원, 최소 CPC 100원**부터 (아이보스 i-boss.co.kr/ab-2987-367796; 실무 평균 CPC 100~500원 — 우강마케팅)
- 광고 소재 문구 3종 A/B (후킹문안 리스트에서 동네 맞춤 변형)
- 광고비는 사장님 계정에서 직접 결제 — 집행비·관리 분리 원칙 명시

### Step 5: reviewer 검증

deliverable-standard 3.5단계 + 과장 문구·허위 혜택 없는지 확인.

## 산출물 (묶음)

```
1. 사장님용 한 장 요약
2. 비즈프로필 세팅 시트 (입력값 미리 채움)
3. 소식 4주분 (날짜 캘린더)
4. 쿠폰 2종 문구
5. 채팅 스크립트 5종
6. 동네 광고 세팅안 (동네 리스트 + 일예산 + 소재 3종)
7. 월간 리포트 틀 (조회/단골/쿠폰/광고 클릭 기록표)
```
