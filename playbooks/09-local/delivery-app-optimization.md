# 배달앱 입점·최적화 플레이북 (음식점 트랙)

## 목적

배달의민족·쿠팡이츠 입점 서류부터 가게소개·메뉴 문안·적법한 리뷰 이벤트까지 **사장님 대신** 준비해 납품한다.

근거: 배민 MAU 2,170만(1위), 쿠팡이츠 1,230만(+32%) (전자신문 2025-11, etnews.com/20251125000090); 서울 카드 결제액은 쿠팡이츠가 배민 추월 (CEO스코어데일리 2025-11) → **양사 병행 입점이 표준**.

## 에이전트 조합

```
copywriter → reviewer
```

## 입력

| 항목 | 설명 | 필수 |
|------|------|------|
| `client` | 클라이언트 폴더명 | 필수 |
| `menu_list` | 전체 메뉴 + 가격 | 필수 |
| `apps` | 입점 대상 (기본: 배민+쿠팡이츠) | 선택 |

## 참조 문서

- `clients/{client}/config.md`, `brand-guidelines.md`
- `prompts/shared/deliverable-standard.md`

## 프레임워크

### Step 1: 입점 서류 체크리스트

공통 4종 + 반려 사전 점검:
- 사업자등록증 / 영업신고증 (**상호·주소가 사업자등록증과 일치해야 함** — 불일치가 반려 1위, baeminceo.com 가이드) / 대표자 본인 명의 통장사본 / 메뉴판 사진 (없으면 메뉴·가격을 종이에 적어 촬영으로 대체 가능)
- 배민: 승인 2~3일, 이후 배민셀프서비스에서 관리 (공식: ceo.baemin.com/guide/4024)
- 쿠팡이츠: partners.coupangeats.com/onboarding 신청, 심사 2~5일, 운영은 store.coupangeats.com (공식 온보딩 + richinsighthub 2025 정리)
- 수수료 투명표: 양사 모두 매출 규모별 2.0~7.8% 차등 (배민 개편 — 톱데일리 topdaily.kr/articles/101680; 쿠팡이츠 2025-04 — 미트인사이트) + 배달비·결제수수료 별도임을 명시

### Step 2: 가게소개·주문안내 (배민 필수 기입란 대응)

- 가게소개문 500자 (배민 신규 가게는 가게소개·주문안내 필수 — 공식: woowahan.zendesk.com/hc/ko/articles/4416122617241)
- 주문안내문 (조리 시간, 요청사항 처리 원칙, 포장 안내)
- 사장님 공지 3종 (첫인사 / 이벤트 / 조리 철학)

### Step 3: 메뉴 문안 전량 리라이팅 + 샷리스트

- 메뉴당 40~60자, 판매 포인트 1개씩 (재료·조리법·양 중 하나)
- 사진 촬영 샷리스트 — **메뉴 이미지 가이드 위반(연출 과장)이 반려 사유**이므로 규격 반영 (공식: woowahan.zendesk.com/hc/ko/articles/4415868277273)

### Step 4: 적법한 리뷰 이벤트 설계 (법규 가드레일)

배달앱 음식점의 67.1%가 리뷰이벤트 운영 (한국소비자원 2024 배달플랫폼 리뷰 실태조사) — 하되 **적법하게**:
- **대가 제공 사실 고지 문구 포함** — 미표시 리뷰는 표시광고법·추천보증 심사지침 위반 소지, 공정위 지속 제재 (easylaw.go.kr 뒷광고 규제; 최재윤 변호사 brunch.co.kr/@jaeyunchoi/36)
- **별점 조건 금지** ("5점 주시면" 류 문구 자동 탈락)
- 방식: 0원/500원 "리뷰이벤트 메뉴" 별도 등록이 통용되는 방식 (pickitsmart.com/baemin-review-event/)
- 리뷰는 배달 후 3일 이내 작성 구조에 맞춰 안내 (9jadu.com)

### Step 5: 리뷰 답글 템플릿 10종

- 별점 5 / 4 / 3 / 1점별 답글 + 배달 지연·누락 사과문
- 욕설·비방 등 정책 위반 리뷰는 신고 절차 안내 (신고 시 삭제·제재 가능)

### Step 6: reviewer 검증

deliverable-standard 3.5단계 + **별점 조건부·대가 미고지 문구 발견 시 즉시 FAIL**.

## 산출물 (묶음)

```
1. 사장님용 한 장 요약
2. 입점 서류 체크리스트 (배민/쿠팡이츠 + 불일치 점검표 + 수수료 투명표)
3. 가게소개 500자 + 주문안내 + 공지 3종
4. 메뉴 문안 전량 + 촬영 샷리스트
5. 적법 리뷰 이벤트 설계안 1종 (고지 문구 포함)
6. 리뷰 답글 템플릿 10종
7. 월간 리포트 틀 (주문 수/리뷰 평점/재주문율 기록표)
```
