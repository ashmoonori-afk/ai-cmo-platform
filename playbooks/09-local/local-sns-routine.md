# 로컬 SNS 최소 루틴 플레이북 (채널 진단 + 주간 루틴)

## 목적

3개 채널을 열어놓고 전부 방치하는 실패를 막는다. **타겟 연령×업종으로 주력 1채널만 고르고**, 사장님이 지킬 수 있는 최소 루틴과 한 달 치 콘텐츠 소재를 만들어 납품한다.

근거 (한국언론진흥재단 2024 소셜미디어 이용자 조사, kpf.or.kr):
- 이용률: 유튜브 84.9% > 인스타그램 38.6% > 밴드 28.6% > 네이버 블로그 21.7%
- 연령 격차: 인스타는 20대 80.9%·30대 70.7%, 50대는 밴드 40.6%·60대 밴드 31.1%
- 숏폼 소비: 유튜브 쇼츠 93.5% ≫ 릴스 36.9%
- **시사점: 고객이 5060이면 네이버·카카오·밴드 우선, 인스타는 2030 유입용.**

## 에이전트 조합

```
strategist → copywriter → reviewer
```

## 입력

| 항목 | 설명 | 필수 |
|------|------|------|
| `client` | 클라이언트 폴더명 | 필수 |
| `owner_capacity` | 사장님이 낼 수 있는 시간 (주당 시간) | 필수 |
| `photo_video_ok` | 촬영 가능 여부 (본인/직원/불가) | 선택 |

## 참조 문서

- `clients/{client}/config.md`, `consultation-notes.md` (팀 규모·여력)
- `outputs/{client}/strategy/*channel-mix*` — 채널믹스 결과 (있으면)
- `prompts/shared/deliverable-standard.md`

## 프레임워크

### Step 1: 채널 선택 진단표 1장 (strategist)

타겟 연령 × 업종 × 사장님 여력 → **주력 1채널 + 보조 0~1채널** 추천 + 근거.
- 고객 2030 + 비주얼 업종(음식/뷰티/공방) → 인스타 릴스
- 고객 4050+ + 검색성 업종(수리/학원/부동산) → 네이버 블로그
- 고객 5060 중심 → SNS보다 플레이스·카카오채널·밴드 우선 (SNS는 보류 권고 가능)
- 주당 여력 2시간 미만 → 채널 1개만, 루틴 축소판

### Step 2: 주간 최소 루틴 카드 (지킬 수 있는 양)

- 인스타 릴스 기준: **주 3회 일관 게시**가 주 1회 완벽주의보다 유리 (linkfarm.ai 2026 데이터 블로그 — 플랫폼 공식 권장치 아님), 게시 시간대 화~금 오전 9시~오후 1시 (같은 출처)
- 포맷 역할: 릴스 = 신규 유입 / 피드 = 아카이브 / 스토리 = 기존 팔로워 소통 (소상공인연합신문 winwinnews.kr/news/877496)
- 네이버 블로그·유튜브 쇼츠의 최적 발행 빈도 실증치는 **미확보** — 주 1~2회로 시작하고 KB로 축적 [실험 항목]
- 한 개 원본 영상 → 릴스+쇼츠 동시 게재로 제작 1회화 [추정 — 도구상 가능, 효과 실증 미확보. 타 플랫폼 워터마크는 반드시 제거 — 인스타 공식 노출 축소 정책]
- **배칭 표준 루틴** (2026-07 리서치 — docs/research/2026-07-02-sns-content-research.md):
  주 1회 60~90분 제작일 (소재 3개 촬영·편집·캡션·예약 등록, 인스타 공식 무료 앱 Edits 활용)
  + 매일 아침 5~10분 (Threads/X 대화형 1줄) + 매일 저녁 10~15분 (댓글·DM 답글 + 스토리 1~2개)
  + 주 1회 10분 인사이트 확인 (도달 대비 공유·완주율 상위 소재 → 다음 주 복제)
- **모든 댓글 24시간 내 답글 규칙**: 답글 시 인게이지 리프트 Threads +42%·인스타 +21% (Buffer 22만 계정, 2026)
- **캡션 첫 줄에 지역+업종 검색 키워드** (예: "○○동 네일 유지력") — 2025-07부터 인스타 공개 프로페셔널 계정이 구글에 인덱싱되어 검색 유입 자산이 됨
- **월 1회 콜라보 게시물**: 이웃 상권 계정·단골 고객과 공동 저자(최대 5계정) — 양쪽 팔로워 피드에 동시 노출되는 무료 신규 도달 수단

### Step 3: 한 달 치 소재 + 완성 문안 (copywriter)

- 촬영 소재 12개 목록 (월분: 만드는 과정 / 손님 반응 / 사장님 이야기 / 계절)
- 회당 15초 스크립트 + 자막 문구 (릴스 기준, 촬영 각도 지시 포함)
- 인스타 프로필 세팅: 바이오 150자 + 하이라이트 커버 명칭 5개
- 해시태그 세트 3벌 (지역+업종+니치, 벌당 15개)
- 블로그 선택 시: 플레이스 연동용 포스트 템플릿 (제목 공식 + 본문 구조 + 사진 배치 지시 — 블로그는 플레이스·검색 유입 목적: 스토어아트 매거진)

### Step 4: reviewer 검증

deliverable-standard 3.5단계 + 루틴이 `owner_capacity`를 초과하지 않는지 확인 (초과 = WARN, 축소판 제시).

## 산출물 (묶음)

```
1. 사장님용 한 장 요약 (내 주력 채널은 ○○, 주 ○회, 회당 ○분)
2. 채널 선택 진단표 1장
3. 주간 루틴 카드 (요일·시간 포함)
4. 촬영 소재 12개 + 15초 스크립트 + 자막
5. 프로필 세팅 문구 + 해시태그 3벌
6. 월간 리포트 틀 (팔로워/도달/문의 기록표)
```
