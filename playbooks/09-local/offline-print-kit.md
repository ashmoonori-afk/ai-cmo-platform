# 오프라인 인쇄물 키트 플레이북 (전단·현수막·간판·명함)

## 목적

시니어 고객층 상권에서 여전히 유효한 오프라인 접점(전단/현수막/간판/명함)의 문안과 **합법 실행 절차**를 사장님 대신 준비한다. 모든 인쇄물의 QR은 네이버 플레이스·카카오채널로 연결해 온라인 채널과 잇는다.

근거: 60대 인터넷 이용률 87.5%로 젊은 층보다 낮고 70대 이상 급락 → 고연령 상권일수록 오프라인 접점 의존 (한국보건사회연구원 2020, repository.kihasa.re.kr); 배달음식점 광고매체 기여도에서 상가 책자·전단이 3위 (요기요 파트너 백서); 소규모 지역 비즈니스에 오프라인 마케팅 유효 (디지털인사이트 ditoday.com).

## 에이전트 조합

```
copywriter → designer → reviewer
```

## 프레임워크

### Step 1: 합법성 체크리스트 (가장 먼저 — 과태료 방지)

**옥외광고물법 핵심**: 현수막·벽보·전단 표시/설치는 지자체 **신고 필요** (공식: easylaw.go.kr 옥외광고물 편). 미신고·지정 외 장소 = 단속 대상.
- 과태료 실례 (성남시 수정구 고지): 도로변 불법 현수막 장당 24만 원, 벽보 2.5만 원, 전단 장당 1.8만 원 — 여러 장 합산 시 500만 원 초과 가능 (sujeong-gu.go.kr 고지)
- 합법 경로 = **지정게시대**: 대체로 전월 초 다음 달분 접수(추첨/선착순), 게시 기간 서울 15일·경기/지방 7일 수준 (1644-9979.co.kr/service/designation-period) — `region` 지자체의 접수 일정·창구를 조사해 표로 제공
- 간판 교체는 규격·허가/신고 대상 구분 확인 (easylaw.go.kr) + 지자체 간판개선사업 신청 가능 여부 안내 (행안부 mois.go.kr 간판개선사업)
- "전단은 신고 불필요"는 오해 — 초보 실수 1위

### Step 2: 전단 A5 앞뒤 카피 (copywriter)

- 헤드라인 3안 (후킹문안 Top 5에서 오프라인 규격 15~25자로 변형)
- 뒷면: 메뉴/서비스 요약 + 쿠폰란 + 약도 + **QR 2개** (네이버 플레이스 / 카카오채널 친구 추가)
- 배포 지침: 음식점은 퇴근 시간대 배포 (오프린트미 ohprint.me 블로그), 아파트 게시판 1주 부착형 등 관리형 채널 우선 (포스트프로모션)

### Step 3: 현수막·간판·명함 문안

- 현수막 문구 3안 — 지정게시대 규격 기준, 멀리서 3초 안에 읽히는 글자 수
- 간판 문안·업종 표기 개선안 (간판 비용 참고: LED 채널 글자당 5~15만 원 + 설치·전기공사 별도 — 매장오픈연구소 openstorelap.co.kr)
- 명함 겸 쿠폰카드 문안 (10회 도장식) — 효과 실증 출처 없음, **저비용 실험으로만** 제안하고 결과를 KB에 기록

### Step 4: 디자인 지시서 (designer)

브랜드 킷의 컬러·글꼴 적용, 인쇄 규격(A5/현수막 표준 규격) + 여백·최소 글자 크기(시니어 가독성: 본문 최소 12pt 권장 [추정 — 인쇄 실무 관례]) 명시.

### Step 5: reviewer 검증

deliverable-standard 3.5단계 + **신고 절차 안내 누락 시 FAIL** (과태료 직결).

## 산출물 (묶음)

```
1. 사장님용 한 장 요약
2. 합법성 체크리스트 (지자체 신고·게시대 일정·과태료 표)
3. 전단 A5 앞뒤 카피 + 배포 지침
4. 현수막 3안 + 간판 개선안 + 명함/쿠폰카드 문안
5. 디자인 지시서 (규격·컬러·글꼴)
6. 인쇄 발주 사양서 (수량·용지·부수별 예산 범위)
```
