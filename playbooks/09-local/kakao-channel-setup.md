# 카카오톡 채널 세팅·단골 관리 플레이북

## 목적

"오프라인 단골을 온라인으로" — 카카오톡 채널을 개설하고 단골 재방문 메시지 체계를 **사장님 대신** 만들어 납품한다. 신규 유입 채널이 아니라 **재방문·재구매(CRM) 채널**이다.

근거: 카카오톡 MAU 4,819만 (와이즈앱 2025-08, wowtale.net/2025/09/24/247595/), 이용률 98.9%로 전 연령 1위 — 50-60대 고객에게도 확실히 닿는 유일한 메시징 채널 (한국언론진흥재단 2024 소셜미디어 이용자 조사).

## 에이전트 조합

```
copywriter → reviewer
```

## 입력

| 항목 | 설명 | 필수 |
|------|------|------|
| `client` | 클라이언트 폴더명 | 필수 |
| `friend_base` | 현재 단골 규모 감 (하루 손님 수 등) | 선택 |
| `offer_cycle` | 재방문 주기 감 (예: 미용실 4주) | 선택 |

## 프레임워크

### Step 0: 무상 지원 먼저 (돈 쓰기 전에)

**카카오 프로젝트 단골**: 연매출 10억 이하 개인/법인 대상 메시지 발송 **무상캐시 30만 원** (유효 1년, 신청 후 최대 1개월 내 결과 — 공식: kakaobusiness.gitbook.io/main/partner/smb). 신청 체크리스트를 산출물에 포함한다. 2022년 이후 6.6만 명이 신청한 검증된 제도 (economidaily.com/view/20241217093858921, 2024).

### Step 1: 채널 개설 시트

절차(공식: kakaobusiness.gitbook.io/main/channel/start, 실무: blog.bizgo.io 2026 가이드):
business.kakao.com 로그인 → 새 채널 만들기 → 사업자등록번호 → 서류 제출 → 심사 1~3영업일.

산출물에 미리 채워 주는 입력값:
- 채널 이름 / 검색용 아이디 후보 3개 / 카테고리 / 소개문 90자 / 프로필 이미지 사양(640×640)
- 서류 제출 시 **주민번호 뒷자리·개인 연락처 마스킹** 경고 포함
- 개설 후 필수 3종: ① 공개 설정 + 검색 허용 ON (안 켜면 검색에 안 뜸 — 초보 실수 1위) ② 고객센터 연락처 등록 ③ 웰컴 메시지 설정

### Step 2: 메시지 문안 세트 (copywriter)

- 웰컴 메시지 1종 + 자동응답 5종 (영업시간/가격/예약/위치/부재중)
- 월간 메시지 캘린더: 월 2회 × 3개월 (쿠폰 / 신메뉴·신상품 / 시즌 인사)
- **건당 비용 계산표 동봉**: 일반 15원/건, 타겟 20원/건, VAT 별도 (공식: cs.kakao.com/helps_html/1073188059) — 예: 친구 1,000명 × 월 2회 ≈ 월 3만~4만 원 [산술 계산]
- 발송 빈도 공식 권장치는 미확보 — 월 2회로 시작하고 수신거부율을 KB에 기록해 조정 [실험 항목]

### Step 3: 친구 모으기 오프라인 키트

매장 접점에서 친구 추가를 유도하는 문구 세트 (즉시 인쇄 가능한 문안):
- 테이블 텐트 / 계산대 POP / 영수증 하단 안내 문구
- 각 문구에 첫 추가 혜택 명시 (예: "친구 추가하면 오늘 음료 500원 할인")
- QR 배치 지시문 (네이버 플레이스·전단과 연동 — 채널 간 연결 설계)

### Step 4: reviewer 검증

- deliverable-standard 3.5단계 (즉시 발송 가능 상태인가, 비용 표기가 투명한가)
- 수신 동의·야간 발송 금지 등 기본 규정 준수 문구 확인

## 산출물 (묶음)

```
1. 사장님용 한 장 요약
2. 무상캐시 30만 원 신청 체크리스트
3. 채널 개설 시트 (입력값 미리 채움, 화면 단위 스텝)
4. 웰컴+자동응답 6종
5. 월간 메시지 캘린더 3개월분 + 비용 계산표
6. 친구 모으기 오프라인 키트 문안 3종
7. 월간 리포트 틀 (친구 수/발송/쿠폰 사용 기록표)
```
