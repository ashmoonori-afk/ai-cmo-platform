# 0단계: 무료 지원 신청 + 대행 사기 감별 플레이북

## 목적

**돈 쓰기 전에 받을 수 있는 것부터.** 정부·플랫폼의 소상공인 마케팅 지원을 신청 가능한 것 위주로 골라 신청 준비물까지 채워 주고, 시니어 사장님을 노리는 광고대행 사기를 감별하는 체크리스트를 제공한다. launch-pack 실행 전 0단계로 권장.

근거: 공정거래위원회가 소상공인 대상 온라인 광고대행 기만영업 업체를 2026-05 기준 18개까지 확대 수사의뢰 — "상위노출 보장", "연매출 상승 보장"이 대표 미끼 (kpinews.kr/newsView/1065598401008816, 2026). 온라인 광고 분쟁 상담·조정 2021년 7,549건 → 2023년 10,452건 (중기부 자료, 한국경제 2024 인용).

## 에이전트 조합

```
researcher → reporter
```

| 순서 | 에이전트 | 역할 | 모델 |
|------|---------|------|------|
| 1 | researcher | 클라이언트 지역·업종·연령 기준 신청 가능 지원사업 조회 (WebSearch로 최신 공고 확인) | sonnet |
| 2 | reporter | 신청 체크리스트 + 사기 감별표를 사장님용으로 정리 | sonnet |

### Step 2: 신청 체크리스트 작성 (reporter)

선정된 사업마다: 자격 요건 확인표 / 준비 서류 목록 / 신청 창구·기간 / 화면 단위 신청 스텝. 마감·미확인 항목은 "[확인 필요 — 공고 링크]"로 정직하게 표기.

### Step 3: 대행 사기 감별표 (필수 동봉)

사장님이 대행 영업 전화를 받았을 때 쓰는 1장짜리 감별표:

**즉시 의심 신호 (하나라도 해당하면 계약 보류):**
- "네이버(카카오) 공식 담당자"라며 전화 — 플랫폼은 영업 전화를 하지 않음 (사칭 판례: 소상공인 700명·7억 편취 — consumernews.co.kr/articleView.html?idxno=648276)
- "상위노출 보장", "파워등급 보장", "연매출 ○○ 보장" — 공정위가 지목한 대표 미끼 문구 (kpinews 2026, 위 출처)
- 월 2~6만 원 저가로 시작하는 1~3년 장기계약 (실제 청구는 수십만 원 할부 — 뉴스쿡 사례; 아이보스 경고 i-boss.co.kr/ab-6141-62956)
- 계약서 없이 구두 안내만, 또는 해지 시 과도한 위약금

**정상 시세 기준표 (2026 열람 기준 — 비교용):**
- 플레이스 관리 월 10만~130만 (등급별) / SNS 운영 월 35만~100만 / 블로그 패키지 월 30만~200만 / 종합 대행 월 30만~200만 (광고비 별도)
- 출처: 트리플아이 pricing, 크몽 견적 가이드, 로카포스팅 시세, 토스페이먼츠 블로그
- 원칙: **광고 집행비는 사장님 계정에 직접 충전** (대행사가 통째로 받아가는 구조는 의심)

### Step 4: reviewer 생략 (검증 면제 아님 — reporter가 최종 정리)

이 SOP 산출물은 안내문이므로 reviewer 게이트 대신 reporter가 출처 링크 유효성을 확인한다. 단 "보장" 류 표현이 산출물에 들어가면 안 된다 (deliverable-standard 9번).
