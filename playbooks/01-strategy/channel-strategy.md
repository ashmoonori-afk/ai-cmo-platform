# 채널 전략 플레이북

## 목적

클라이언트가 활용 중이거나 활용할 수 있는 모든 채널을 체계적으로 평가하고, 채널별 목적·콘텐츠 유형·운영 빈도·KPI를 정의한다. 채널 간 시너지 맵을 통해 각 채널이 퍼널 내에서 어떤 역할을 하고 어떻게 연결되는지 명확히 한다. 리소스가 한정된 팀이 집중할 채널과 제외할 채널을 결정하는 기준을 제공한다.

## 에이전트 조합

```
[1단계] researcher   →  채널별 업종 평균 성과 데이터 수집
[2단계] strategist   →  채널 매트릭스 작성 + 시너지 맵 설계
[3단계] reviewer     →  채널 선택 근거·실행 가능성 검증
```

**실행 명령 예시:**
```
"[클라이언트명] 채널 전략 수립.
researcher: {업종}의 채널별 평균 CTR, CPA, 전환율 벤치마크 데이터 수집.
strategist: 채널 매트릭스 + 시너지 맵 작성.
reviewer: 팀 규모({X}명), 예산(₩{X}) 기준 실행 가능성 확인."
```

## 입력

| 항목 | 설명 | 필수 여부 |
|------|------|----------|
| 클라이언트 config | `clients/{client}/config.md` — ICP, 업종, 현재 채널 현황 | 필수 |
| GTM 모션 분석 | `gtm-motion-analysis` 출력물 — 우선 모션 3개 | 권장 |
| 팀 규모 & 예산 | 마케팅 팀 인원, 월 예산 | 필수 |
| 현재 채널 성과 | 채널별 현재 트래픽·전환·비용 데이터 | 권장 |
| 경쟁사 채널 현황 | competitor 에이전트 수집 데이터 | 권장 |

## 참조 문서

- `clients/{client}/config.md` — 클라이언트 기본 설정
- `playbooks/01-strategy/gtm-motion-analysis.md` — GTM 모션 우선순위
- `playbooks/01-strategy/funnel-design.md` — 퍼널 단계별 채널 연결
- `prompts/shared/boilerplate.md` — 컨텍스트 로딩 규칙

## 프레임워크

### Step 1. 채널 인벤토리 작성

분석 대상 채널을 소유(Owned)·유료(Paid)·획득(Earned) 미디어로 분류한다:

**Owned Media (자사 소유 채널):**
- 웹사이트 / 블로그
- 이메일 뉴스레터
- SNS 계정 (LinkedIn, Instagram, Twitter/X, 카카오, 네이버 블로그)
- 앱 푸시 알림

**Paid Media (비용 지불 채널):**
- 검색 광고 (네이버 파워링크, Google Ads)
- SNS 광고 (Meta, LinkedIn, 카카오 비즈보드)
- 디스플레이 / 배너
- 인플루언서 마케팅

**Earned Media (획득 채널):**
- 자연 검색 (SEO)
- 바이럴 / 공유
- PR / 미디어 기사
- 고객 리뷰 / 추천

---

### Step 2. 채널 매트릭스

각 채널을 7개 기준으로 평가한다:

| 채널 | 퍼널 목적 | 콘텐츠 유형 | 운영 빈도 | 주요 KPI | 현재 성과 | 업종 벤치마크 | 투자 우선순위 |
|------|---------|-----------|---------|---------|---------|------------|------------|
| **[Owned]** | | | | | | | |
| 웹사이트 블로그 | 인지/관심 | 롱폼 콘텐츠 | 주 2회 | 방문자, 체류시간 | | | |
| 이메일 뉴스레터 | 관심/전환 | 큐레이션 + CTA | 격주 1회 | 오픈율, CTR | | | |
| LinkedIn 페이지 | 인지/관심 | 인사이트 포스트 | 주 3-5회 | 팔로워, 인게이지먼트 | | | |
| Instagram | 인지 | 비주얼 콘텐츠 | 주 3회 | 팔로워, 저장 | | | |
| 네이버 블로그 | 인지/관심 | SEO 콘텐츠 | 주 2-3회 | 방문자, 검색 순위 | | | |
| **[Paid]** | | | | | | | |
| 네이버 검색 광고 | 전환 | 텍스트 광고 | 상시 | CPC, ROAS | | | |
| Google Ads | 전환 | 텍스트/배너 | 상시 | CPC, CVR | | | |
| 카카오 비즈보드 | 인지/관심 | 이미지 광고 | 시즌 집중 | CPM, CTR | | | |
| Meta 광고 | 인지/리드 | 이미지/동영상 | 캠페인 기간 | CPL, ROAS | | | |
| **[Earned]** | | | | | | | |
| SEO (자연 검색) | 인지/관심 | 블로그·FAQ | 지속 누적 | 키워드 순위, 오가닉 트래픽 | | | |
| 고객 리뷰/추천 | 전환/유지 | UGC | 지속 수집 | 리뷰 수, NPS | | | |

**퍼널 목적 범례:**
- 인지 (Awareness): 브랜드 최초 노출
- 관심 (Interest): 콘텐츠 소비·팔로우
- 고려 (Consideration): 비교·평가 단계
- 전환 (Conversion): 구매·가입·문의
- 유지 (Retention): 재구매·재방문
- 추천 (Referral): 지인 공유·리뷰

---

### Step 3. 채널 성과 평가 & 우선순위 결정

**채널별 점수 산정 (각 10점):**

| 채널 | ROI 잠재력 | 실행 난이도 (낮을수록 좋음) | 타겟 도달력 | 종합 점수 | 투자 결정 |
|------|----------|----------------------|----------|---------|--------|
| | /10 | /10 | /10 | /10 | 집중/유지/축소/중단 |

**투자 결정 기준:**
- **집중 (Scale Up):** 종합 점수 7점 이상, 현재 성과 목표 대비 달성 중
- **유지 (Hold):** 종합 점수 5-7점, 일정 성과 유지 중
- **개선 (Optimize):** 종합 점수 5점 미만이지만 전략적으로 필요
- **중단 (Cut):** 종합 점수 5점 미만 + 성과 부진

### Step 5. 채널 간 시너지 맵

채널들이 퍼널 내에서 어떻게 연결되고 상호 보완하는지 시각화한다:

**시너지 맵 (텍스트 다이어그램):**

```
[인지 단계]
  네이버 검색 광고  →  (클릭)  →  블로그 / 웹사이트
  SNS 광고          →  (클릭)  →  랜딩 페이지
  SEO 자연 검색     →  (방문)  →  블로그 / 제품 페이지
       |
       | (이탈 → 리타겟팅)
       v
[관심 단계]
  블로그 콘텐츠     →  (구독 유도)  →  이메일 뉴스레터
  LinkedIn 포스트  →  (팔로우)     →  지속 콘텐츠 노출
       |
       v
[고려/전환 단계]
  이메일 시퀀스     →  (CTA)  →  데모 예약 / 상품 페이지
  리타겟팅 광고     →  (CTA)  →  랜딩 페이지
       |
       v
[유지/추천 단계]
  이메일 (온보딩)   →  재방문
  고객 리뷰 수집    →  SEO + 신규 인지 루프
```

**채널 간 데이터 전달 규칙:**
- 블로그 방문자 → 이메일 구독 팝업 (관심 포착)
- 광고 클릭 → 픽셀/태그 심어 리타겟팅 풀 구성
- 이메일 CTA → UTM 파라미터로 전환 추적
- 구매 완료 → 리뷰 요청 이메일 자동 발송

---

### Step 6. 채널 운영 루틴 (Weekly Checklist)

**월요일:** 지난 주 채널별 성과 리뷰 (KPI 대시보드 확인)
**화요일:** 이번 주 발행할 콘텐츠 최종 확인 및 예약
**수요일:** 이메일 발송 (격주)
**목요일:** SNS 인게이지먼트 관리 (댓글·DM 응답)
**금요일:** 광고 성과 확인 (CPC, ROAS 이상값 체크)

**월간 루틴:**
- 매월 1일: 채널별 KPI 리포트 작성 (`06-analytics` 플레이북 참조)
- 매월 마지막 주: 다음 달 콘텐츠 캘린더 작성
- 분기별: 채널 우선순위 재평가 (이 플레이북 재실행)

---

### Step 7. 채널 실험 프레임워크

새 채널을 테스트할 때 적용하는 기준:

**실험 조건:**
- 테스트 기간: 최소 4주
- 최소 예산: 월 ₩{X} (의미 있는 데이터 수집을 위한 최소값)
- 성공 기준: 해당 채널에서 {X}건 이상 전환 또는 CPA ₩{X} 이하

**실험 종료 기준 (이 중 하나라도 해당 시 중단):**
- 4주 후 전환 0건
- CPA가 목표 대비 3배 초과
- 채널 특성이 ICP와 불일치 (정성적 판단)
