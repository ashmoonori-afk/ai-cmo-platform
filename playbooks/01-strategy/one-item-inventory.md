# 일품 재고 마케팅 전략 플레이북

## 목적
빈티지/구제, 핸드메이드, 한정판 등 모든 상품이 1점뿐인 비즈니스의 마케팅 전략을 수립한다. 일반 재고 기반 마케팅과 근본적으로 다른 접근이 필요.

## 에이전트 조합
strategist → copywriter → reviewer

## 입력
- client, product_category, avg_inventory_count (평균 보유 재고 수), turnover_rate (회전율)

## 프레임워크

### 일품 재고 비즈니스의 특수성
| 일반 비즈니스 | 일품 재고 비즈니스 |
|-------------|----------------|
| 재입고/재주문 가능 | 판매되면 영원히 소멸 |
| 표준 가격표 | 아이템별 개별 가격 |
| 리마케팅 가능 | 리마케팅 시 이미 품절 가능 |
| 재고 관리 | 소싱이 곧 재고 |
| 제품 페이지 영구 | 제품 페이지 수시 소멸 |
| 광고 ROI 측정 용이 | 단일 아이템 광고 ROI 측정 어려움 |

### 핵심 전략: 희소성 마케팅
1. **FOMO(Fear of Missing Out) 활용**: "1점뿐", "재입고 없음" 메시지 일관 적용
2. **신상품 알림 시스템**: 새 소싱 도착 → SNS 즉시 업로드 → 찜/알림 유도
3. **큐레이션 브랜딩**: 개별 아이템이 아닌 "셀렉션 안목" 자체를 브랜드화
4. **카테고리 SEO**: 개별 상품 SEO 불가 → 카테고리/스타일 키워드 SEO로 전환
5. **스토리텔링**: 아이템의 출처/연대/상태를 스토리로 풀어 가치 상승

### 가격 전략 (일품 재고 특화)
- **희소성 프리미엄**: 유니크 아이템일수록 프리미엄 가산
- **컨디션 기반 가격**: S/A/B/C 등급별 가격 차등
- **소싱 원가 기반**: 원가 x 마진율 (일반 2-4배)
- **시즌 반영**: 시즌 아이템은 해당 시즌에 가격 상향

### 콘텐츠 전략 (일품 재고 특화)
- **실시간 콘텐츠**: 신규 입고 즉시 촬영/업로드 (계획 콘텐츠보다 실시간 우선)
- **큐레이션 콘텐츠**: "이번 주 베스트 셀렉션", "에디터 추천"
- **스타일링 콘텐츠**: 개별 아이템 착용/활용 예시
- **비하인드 콘텐츠**: 소싱 여행, 선별 과정 공개 (브랜드 스토리)

### 퍼널 특화
```
[인지] SNS 피드/검색 → [관심] 스토어 방문 → [고려] 상품 상세 확인
→ [긴급성] "1점뿐" FOMO → [전환] 즉시 구매 (고려 시간 짧음)
→ [유지] 신상 알림 구독 → [재방문] 신규 입고 시 재방문
```
