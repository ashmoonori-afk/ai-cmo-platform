# 캠페인 기획 플레이북

## 목적

단발성 아이디어가 아닌 구조화된 캠페인 기획서를 생산한다. SMART 목표 설정에서 시작해 ICP 기반 타겟팅, 채널 믹스, 콘텐츠 캘린더, 예산 배분, KPI 대시보드까지 캠페인 실행에 필요한 모든 요소를 하나의 문서에 담는다. 기획자·실행자·리뷰어가 같은 문서를 보며 정렬될 수 있게 하는 것이 핵심이다.

## 에이전트 조합

```
[1단계] strategist   →  목표 설정 + 타겟 정의 + 채널 믹스 설계
[2단계] reviewer     →  목표 현실성·채널 적합성·예산 배분 검증
```

**실행 명령 예시:**
```
"[클라이언트명] {캠페인 이름} 기획.
목표: {예: Q2 신규 가입자 500명, 리드 200건}
기간: {YYYY-MM-DD ~ YYYY-MM-DD}
예산: ₩{X}
strategist: 위 조건으로 캠페인 기획서 초안 작성.
reviewer: 목표·채널·예산 정합성 검토."
```

## 입력

| 항목 | 설명 | 필수 여부 |
|------|------|----------|
| 클라이언트 config | `clients/{client}/config.md` — ICP, 경쟁사, 채널 현황 | 필수 |
| 캠페인 목적 | 인지 확대 / 리드 생성 / 전환 / 재구매 / 브랜드 중 택1 | 필수 |
| 예산 | 총 캠페인 예산 (₩ 또는 $ 단위로 명시) | 필수 |
| 기간 | 캠페인 시작일 ~ 종료일 | 필수 |
| 현재 베이스라인 | 현재 월간 가입자·리드·매출 수치 (목표 설정 기준) | 권장 |
| 포지셔닝 스테이트먼트 | `positioning-map` 출력물 또는 config.md의 핵심 메시지 | 권장 |

## 참조 문서

- `clients/{client}/config.md` — ICP, 가치 제안, 채널 현황
- `playbooks/01-strategy/positioning-map.md` — 메시지 프레임워크
- `playbooks/01-strategy/channel-strategy.md` — 채널별 특성 참조
- `prompts/shared/boilerplate.md` — 컨텍스트 로딩 규칙

## 프레임워크

### Step 1. 캠페인 개요 (Campaign Brief)

캠페인을 한 장에 요약하는 브리프를 먼저 작성한다:

| 항목 | 내용 |
|------|------|
| 캠페인명 | |
| 기간 | {YYYY-MM-DD} ~ {YYYY-MM-DD} ({X}주) |
| 총 예산 | ₩{X} |
| 캠페인 목적 | 인지 / 리드 생성 / 전환 / 재구매 / 브랜드 |
| 핵심 타겟 | ICP 세그먼트 {A, B} |
| 핵심 메시지 | 포지셔닝 스테이트먼트에서 발췌한 1문장 |
| 주요 채널 | {채널 1, 2, 3} |
| 성공 기준 | KPI 목표 1~2개 |

---

### Step 2. SMART 목표 설정

캠페인의 모든 결정은 SMART 목표에서 역산된다.

**SMART 기준:**
- **S**pecific (구체적): 숫자로 표현된 목표
- **M**easurable (측정 가능): 추적 방법이 명확
- **A**chievable (달성 가능): 베이스라인 대비 합리적 성장률
- **R**elevant (관련성): 비즈니스 핵심 목표와 연결
- **T**ime-bound (기한): 명확한 달성 기한

**목표 설정 워크시트:**

| 레벨 | 목표 유형 | SMART 목표 | 베이스라인 | 목표값 | 성장률 |
|------|---------|-----------|---------|------|------|
| 비즈니스 | 매출 / GMV | | ₩{X}/월 | ₩{X} | +{X}% |
| 마케팅 | 리드 / 가입 | | {X}건/월 | {X}건 | +{X}% |
| 채널 | 트래픽 / 노출 | | {X}/월 | {X} | +{X}% |

**목표 합리성 검증:**
- 과거 12개월 월평균 성장률: {X}%
- 이번 캠페인 목표 성장률: {X}%
- 추가 예산 투입 대비 기대 성장: {근거 제시}

---

### Step 3. 타겟 정의 (ICP 기반)

**주요 타겟 세그먼트:**

| 세그먼트 | 특성 (인구통계·행동) | 핵심 페인포인트 | 이 캠페인과 접점 | 예상 비중 |
|--------|-----------------|-------------|--------------|--------|
| 세그먼트 A | | | | {X}% |
| 세그먼트 B | | | | {X}% |

**타겟 우선순위 기준:**
- 전환 가능성이 가장 높은 세그먼트: {세그먼트 A} — 이유: {}
- 이번 캠페인에서 제외하는 세그먼트: {} — 이유: {}

---

### Step 4. 채널 믹스 설계

**퍼널 단계별 채널 역할 배분:**

```
인지 (Awareness)     →  {채널: 예 - 유료 광고, SNS}
관심 (Interest)      →  {채널: 예 - 블로그, 이메일}
고려 (Consideration) →  {채널: 예 - 케이스스터디, 웨비나}
전환 (Conversion)    →  {채널: 예 - 랜딩 페이지, 세일즈 콜}
```

**채널별 역할·예산·KPI:**

| 채널 | 퍼널 단계 | 예산 | 예산 비중 | 주요 KPI | 목표값 |
|------|---------|------|---------|---------|------|
| 유료 광고 (검색) | 인지/전환 | ₩{X} | {X}% | CPC, 전환율 | |
| 유료 광고 (SNS) | 인지/관심 | ₩{X} | {X}% | CPM, CTR | |
| 이메일 마케팅 | 관심/전환 | ₩{X} | {X}% | 오픈율, 클릭률 | |
| 콘텐츠 (블로그/SNS) | 인지/관심 | ₩{X} | {X}% | 방문자, 체류시간 | |
| 파트너십/제휴 | 전환 | ₩{X} | {X}% | 파트너 발 전환 | |
| **합계** | | **₩{X}** | **100%** | | |

---

### Step 5. 핵심 메시지 & 콘텐츠 계획

**캠페인 핵심 메시지 (포지셔닝 기반):**
- Hero Message: {1문장}
- 채널별 변형 메시지:
  - 검색 광고: {키워드 포함, 30자 이내}
  - SNS: {감성적, 이미지와 병행}
  - 이메일: {개인화 요소 포함}

**콘텐츠 유형별 계획:**

| 콘텐츠 유형 | 수량 | 담당 에이전트 | 참조 플레이북 |
|-----------|------|------------|------------|
| 블로그 포스트 | {X}편 | seo-specialist + copywriter | `03-content/blog-article.md` |
| SNS 포스트 | {X}건 | copywriter | `03-content/social-post.md` |
| 이메일 시퀀스 | {X}통 | copywriter | `03-content/newsletter.md` |
| 광고 카피 | {X}세트 | copywriter | `03-content/ab-copy.md` |
| 랜딩 페이지 | {X}개 | copywriter + reviewer | — |
