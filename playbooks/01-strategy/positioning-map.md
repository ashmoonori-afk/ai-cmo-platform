# 포지셔닝 맵 플레이북

## 목적

경쟁사와 자사를 2x2 매트릭스에 시각적으로 배치하고, 차별화된 포지셔닝 스테이트먼트와 메시지 프레임워크를 도출한다. 시장에서 자사가 소유해야 할 고유한 영역(white space)을 발굴하고, 이를 모든 마케팅 커뮤니케이션의 기준 언어로 만드는 것이 목적이다.

## 에이전트 조합

```
[1단계] competitor   →  경쟁사 포지셔닝 데이터 수집 (웹사이트·카피·리뷰 분석)
[2단계] strategist   →  축 선정 + 2x2 배치 + 포지셔닝 스테이트먼트 작성
[3단계] reviewer     →  스테이트먼트 명확성·차별성·신뢰성 검증
```

**실행 명령 예시:**
```
"[클라이언트명] 포지셔닝 맵 작성.
competitor: 경쟁사 {A, B, C}의 주요 메시지, 타겟, 가격대 수집.
strategist: 수집 데이터로 2x2 맵 + 포지셔닝 스테이트먼트 작성.
reviewer: 스테이트먼트가 구체적이고 검증 가능한지 확인."
```

## 입력

| 항목 | 설명 | 필수 여부 |
|------|------|----------|
| 클라이언트 config | `clients/{client}/config.md` — ICP, 핵심 가치 제안, 경쟁사 목록 | 필수 |
| 브랜드 가이드라인 | `clients/{client}/brand-guidelines.md` — 톤앤매너, 브랜드 퍼스낼리티 | 필수 |
| 경쟁사 조사 결과 | competitor 에이전트 수집 데이터 또는 기존 경쟁사 분석 자료 | 필수 |
| 고객 인사이트 | `knowledge-base/{client}/insights.md` — 고객 구매 동인, VOC | 권장 |

## 참조 문서

- `clients/{client}/config.md` — 클라이언트 기본 설정
- `clients/{client}/brand-guidelines.md` — 브랜드 보이스 & 퍼스낼리티
- `prompts/shared/boilerplate.md` — 컨텍스트 로딩 규칙

## 프레임워크

### Step 1. 차별화 축 선정

포지셔닝 맵의 X축·Y축은 **고객이 구매 결정 시 실제로 중요하게 여기는 기준**이어야 한다.
아래 후보 축에서 업종 특성에 맞는 2개를 선택한다:

**축 후보 목록:**

| 축 유형 | 예시 | 적합 업종 |
|--------|------|---------|
| 가격 (저가 ↔ 고가) | 가격 경쟁력 | 커머스, B2C SaaS |
| 전문성 (범용 ↔ 전문) | 업종 특화도 | B2B 서비스, 컨설팅 |
| 속도 (느림 ↔ 빠름) | 배송·처리 속도 | 물류, 커머스 |
| 개인화 (표준 ↔ 맞춤) | 서비스 맞춤도 | SaaS, 교육 |
| 접근성 (복잡 ↔ 간편) | UX 편의성 | 앱, B2B 툴 |
| 신뢰성 (신생 ↔ 검증됨) | 레퍼런스 · 브랜드 신뢰 | 엔터프라이즈 B2B |
| 커버리지 (로컬 ↔ 전국) | 서비스 지역 범위 | 지역 서비스 |
| 디자인 감성 (기본 ↔ 감성적) | 제품 디자인/미학 수준 | 패션, 라이프스타일, 인테리어 |
| 소재 품질 (합성 ↔ 천연/유기농) | 원재료/소재 품질 | 의류, 식품, 화장품 |
| 트렌디 (클래식 ↔ 트렌디) | 유행 반영 속도 | 패션, 뷰티, 가전 |
| 감성 가치 (실용적 ↔ 감성적) | 기능 vs 감성/경험 중심 | D2C, 라이프스타일, 선물 |

**선택된 축:**
- **X축:** {축 이름} (왼쪽: {낮은 값} / 오른쪽: {높은 값})
- **Y축:** {축 이름} (아래: {낮은 값} / 위: {높은 값})

**축 선택 근거:**
- 왜 이 두 축이 타겟 고객의 의사결정에 가장 중요한가? (2-3줄)

---

### Step 2. 경쟁사 + 자사 배치

각 플레이어를 X·Y 좌표(1-10 척도)로 평가하고 2x2 맵에 배치한다.

**플레이어 좌표표:**

| 플레이어 | X축 점수 (1-10) | Y축 점수 (1-10) | 포지셔닝 한 줄 |
|--------|--------------|--------------|-------------|
| 자사 ({클라이언트명}) | | | |
| 경쟁사 A | | | |
| 경쟁사 B | | | |
| 경쟁사 C | | | |
| (추가) | | | |

**텍스트 기반 2x2 포지셔닝 맵:**

```
Y축 높음 ({Y축 높은 값})
^
|
|   [2사분면]              [1사분면]
|   X낮음 · Y높음         X높음 · Y높음
|
|   {경쟁사 B}            {자사명} ★
|                         {경쟁사 A}
|
+---------------------------------> X축 높음 ({X축 높은 값})
|
|   [3사분면]              [4사분면]
|   X낮음 · Y낮음         X높음 · Y낮음
|
|   {경쟁사 C}
|
Y축 낮음 ({Y축 낮은 값})
```

**화이트 스페이스 (공백 영역) 발굴:**
- 어느 사분면이 경쟁자가 없거나 적은가?
- 그 영역이 고객에게 가치 있는 영역인가?
- **결론:** 자사가 소유해야 할 포지션은 {사분면/좌표} 영역 — {이유 한 줄}

---

### Step 3. 포지셔닝 스테이트먼트

**표준 템플릿:**
```
For [타겟 고객],
[제품/서비스명]은
[핵심 차별점 또는 가장 중요한 혜택]을 제공하는
[카테고리/대안]입니다.
Unlike [주요 대안/경쟁사],
[자사는 {근거가 되는 구체적 사실}를 통해 이를 가능하게 합니다].
```

**작성 기준:**
- 타겟 고객: ICP를 직접 지칭 (직무·업종·상황 포함)
- 핵심 차별점: 경쟁사가 제공하지 못하는 단 한 가지
- 카테고리: 고객이 자사를 어떤 대안과 비교하는가
- 근거: 기능적 사실(수치·특허·프로세스)로 뒷받침

**검증 체크리스트:**
- [ ] 이 스테이트먼트를 읽으면 누가 사야 하는지 즉시 알 수 있는가?
- [ ] 경쟁사가 동일한 스테이트먼트를 주장할 수 없는가?
- [ ] 과장 없이 증명 가능한 사실에 기반하는가?
- [ ] 구매 결정을 유도하는 감성적 울림이 있는가?

---

### Step 4. 메시지 프레임워크

**핵심 메시지 1개 (Hero Message):**
- 단 한 문장으로 자사의 존재 이유를 표현
- 15단어 이내, 타겟 고객의 언어로 작성

**서포팅 메시지 3개 (Pillar Messages):**

| # | 서포팅 메시지 | 뒷받침 근거 | 타겟 페르소나 |
|---|------------|-----------|------------|
| 1 | | | |
| 2 | | | |
| 3 | | | |

**타겟별 메시지 변형:**

| 타겟 세그먼트 | 강조 포인트 | 적합한 채널 |
|------------|---------|-----------|
| 세그먼트 A | | |
| 세그먼트 B | | |
| 세그먼트 C | | |

---

### Step 5. 차별화 근거 정리

포지셔닝을 뒷받침하는 차별화 포인트를 3가지 레이어로 정리한다:

| 차별화 레이어 | 내용 | 고객 인지 용이성 |
|------------|------|--------------|
| **기능적 차별점** (제품·서비스 특성) | | 높음 (쉽게 비교) |
| **감성적 차별점** (느낌·경험) | | 중간 |
| **상징적 차별점** (브랜드 이미지·소속감) | | 낮음 (장기 구축) |

**경쟁사 대비 차별화 매트릭스:**

| 차별화 요소 | 자사 | 경쟁사 A | 경쟁사 B | 경쟁사 C |
|-----------|------|--------|--------|--------|
| {요소 1} | ✅ | ❌ | ✅ | ❌ |
| {요소 2} | ✅ | ✅ | ❌ | ❌ |
| {요소 3} | ✅ | ❌ | ❌ | ❌ |

---

### Step 6. 포지셔닝 적용 가이드

이 포지셔닝을 실제 커뮤니케이션에 적용하는 규칙:

**적용해야 할 것:**
- 웹사이트 히어로 섹션 헤드라인에 핵심 메시지 반영
- 영업 자료 첫 페이지에 포지셔닝 스테이트먼트 포함
- 광고 카피의 모든 변형은 서포팅 메시지 3개 중 하나를 기반으로 작성

**피해야 할 것:**
- "최고의", "최저가", "최다" 등 증명 불가 최상급 표현
- 경쟁사도 동일하게 주장하는 일반적 가치 (예: "신뢰할 수 있는", "빠른")
- 타겟 세그먼트마다 전혀 다른 핵심 메시지 사용 (일관성 훼손)
