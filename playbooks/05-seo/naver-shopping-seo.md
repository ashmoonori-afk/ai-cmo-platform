# 네이버 쇼핑 SEO 플레이북

## 목적
네이버 스마트스토어/쇼핑 검색 알고리즘에 최적화하여 상품 노출을 극대화한다. Google SEO와는 별개의 최적화 체계.

## 에이전트 조합
seo-specialist → reviewer

## 입력
- client, store_url (스마트스토어 URL), product_categories

## 프레임워크

### 네이버 쇼핑 랭킹 요소 (가중치 추정)
| 요소 | 가중치 | 최적화 방법 |
|------|-------|-----------|
| 키워드 적합성 | 30% | 상품명에 핵심 키워드 배치 (브랜드명+카테고리+속성+상세) |
| 최근 판매량 | 25% | 초기 프로모션으로 판매 이력 확보, 묶음 할인 |
| 리뷰 수/평점 | 20% | 구매 확정 후 리뷰 유도, 포토리뷰 인센티브 |
| 찜(좋아요) 수 | 10% | 찜하기 이벤트, SNS 연계 |
| 클릭률(CTR) | 10% | 대표 이미지 품질, 가격 경쟁력, 혜택 뱃지 |
| 스토어 신뢰도 | 5% | 판매자 등급(씨앗~빅파워), 응답률, 교환/반품률 |

### 상품명 최적화 공식
```
{브랜드명} {핵심카테고리} {소재/특성} {용도/시즌} {상세설명}
```
예: "Sample Client D PPSU 젖병 신생아용 160ml 트라이탄 빨대컵 호환"

### 스마트스토어 SEO 체크리스트
- [ ] 상품명 40자 이내, 핵심 키워드 앞배치
- [ ] 카테고리 정확 매핑 (네이버 카테고리 트리)
- [ ] 상품 속성 태그 전부 입력
- [ ] 대표 이미지 640x640 이상, 깔끔한 배경
- [ ] 상세페이지 이미지 10장+, 텍스트 삽입
- [ ] 가격 경쟁력 확인 (동일 상품 최저가 대비)
- [ ] 무료배송 또는 조건부 무료배송 설정
- [ ] 쿠폰/적립금 혜택 뱃지 활성화
- [ ] 리뷰 50개+ 목표 (초기 판매 프로모션)

### 네이버 쇼핑 광고 (파워링크/쇼핑검색광고)
- 키워드 선정: 네이버 키워드 도구 활용
- 입찰가 전략: 순위별 CPC 비교
- 성과 추적: ROAS 기준

## 출력 경로
outputs/{client}/seo/{YYYYMMDD}_naver-shopping-seo.md
