# 소셜 캘린더 플레이북

## 목적

월간 SNS 발행 일정을 체계적으로 계획하고 포스트별 카피 초안을 미리 작성함으로써, 콘텐츠의 일관성과 발행 주기를 확보하고 팀 운영 효율을 높인다. 퍼널 단계(인지 → 관심 → 전환)를 균형 있게 배분하여 채널 전반의 마케팅 목표를 달성한다.

---

## 에이전트 조합

```
strategist → copywriter → reviewer
```

| 순서 | 에이전트 | 역할 | 모델 |
|------|---------|------|------|
| 1 | strategist | 월간 주제·캠페인 기획, 퍼널 배분 설계 | opus |
| 2 | copywriter | 포스트별 카피 초안 작성 | sonnet |
| 3 | reviewer | 일관성·브랜드·퍼널 균형 검증 | sonnet |

---

## 입력

| 항목 | 설명 | 필수 여부 |
|------|------|---------|
| `client` | 클라이언트 폴더명 | 필수 |
| `month` | 대상 월 (예: 2026-04) | 필수 |
| `platforms` | 운영 플랫폼 (기본: LinkedIn, Instagram, Twitter) | 선택 |
| `monthly_theme` | 이달의 마케팅 테마 또는 캠페인 | 선택 |
| `key_events` | 이달 주요 이벤트·기념일·프로모션 | 선택 |
| `post_frequency` | 플랫폼별 주간 발행 횟수 (기본: 주 3회) | 선택 |
| `funnel_ratio` | 인지:관심:전환 비율 (기본: 5:3:2) | 선택 |

## 프레임워크

### Step 1 — strategist: 월간 기획

```
1. 이달의 마케팅 목표 설정 (인지/리드수집/전환 중 우선순위)
2. 월간 테마 1개 선정 (모든 포스트의 통일된 맥락)
3. 주간 서브 테마 설계 (W1/W2/W3/W4)
4. 퍼널 배분 결정 (인지:관심:전환 = 5:3:2 기본)
5. 주요 이벤트·기념일 반영
6. 플랫폼별 발행 일정 배치
```

### Step 2 — copywriter: 포스트별 카피 초안

```
각 포스트에 포함할 정보:
- 플랫폼: LinkedIn / Instagram / X / Threads
- 발행일: YYYY-MM-DD (요일)
- 주제: {주제 한 줄}
- 카피 초안: 50자 내외 (플랫폼별 규격은 social-post.md 준수)
- 목적: 인지 / 관심 / 전환
- 비주얼 방향: 이미지·영상 컨셉 1줄 (선택)
- 해시태그: 3–5개 (분류용 — Threads는 태그 1개)

발행 빈도·시간대 기본값 (2026-07 리서치 반영 — docs/research/2026-07-02-sns-content-research.md):
- LinkedIn: 주 2–3회 (월·수·금)
- Instagram: 주 3–5회 (릴스 2–3 + 캐러셀 1–2) + 스토리 일 1–2
- X: 일 1–2회 (짧은 대화형 포함)
- Threads: 주 3–5회 (일 3회 초과 금지)
- 시간대: 화–수 11–18시 우선, 주말 회피 (글로벌 데이터 — Sprout 2026)
  ※ 반드시 클라이언트 계정 인사이트의 '팔로워 활동 시간'으로 보정 (한국 전용 데이터 없음)
- 쉬는 주 금지: 게시를 쉰 주는 자기 성장 기준선보다 하락 (Buffer 5,200만 게시물, 2026)

트라이얼 릴스 슬롯 (팔로워 1,000명 이상 계정):
- 주 1–2개 소재를 트라이얼 릴스(비팔로워 선노출)로 검증 → 승자만 캘린더 본편성
```

### 퍼널 배분 가이드

```
인지 (Awareness) — 5할:
- 업계 트렌드, 교육 콘텐츠, 흥미로운 통계
- 공유 가능성 높은 포스트
- 브랜드 성격·가치관 보여주기

관심 (Interest) — 3할:
- 제품/서비스 기능 소개
- HOW-TO, 팁, 가이드
- 케이스스터디 요약

전환 (Conversion) — 2할:
- 무료 체험·데모 신청 유도
- 프로모션·할인 공지
- 상담 신청 CTA
```
