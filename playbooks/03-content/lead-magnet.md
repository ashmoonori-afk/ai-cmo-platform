# 리드마그넷 플레이북

## 목적

ICP(이상적 고객)가 이메일·연락처를 제공할 만큼 충분한 가치를 제공하는 무료 콘텐츠 자료(체크리스트, 가이드, 템플릿, 워크시트)를 제작하여 리드를 수집하고, 구독자를 잠재 고객으로 전환한다.

---

## 에이전트 조합

```
strategist → copywriter → reviewer
```

| 순서 | 에이전트 | 역할 | 모델 |
|------|---------|------|------|
| 1 | strategist | 유형 선택·목차 설계·ICP 관련성 검토 | opus |
| 2 | copywriter | 리드마그넷 전체 원고 작성 (PDF 변환용 마크다운) | sonnet |
| 3 | reviewer | 구조·브랜드·가치 제공 충분성 검증 | sonnet |

---

## 입력

| 항목 | 설명 | 필수 여부 |
|------|------|---------|
| `client` | 클라이언트 폴더명 | 필수 |
| `topic` | 리드마그넷 주제 | 필수 |
| `type` | 유형 (checklist / guide / template / worksheet) | 선택 (strategist가 선정) |
| `icp_pain_point` | 대상 ICP의 핵심 고충 | 선택 |
| `funnel_stage` | 퍼널 단계 (awareness / consideration / conversion) | 선택 |
| `cta_offer` | 풀 버전 또는 다음 단계 제안 내용 | 선택 |
| `page_count` | 목표 페이지 수 (기본: 5–10페이지) | 선택 |

---

| 문서 | 경로 | 목적 |
|------|------|------|
| 클라이언트 설정 | `clients/{client}/config.md` | ICP·제품·업종 파악 |

## 프레임워크

### Step 1 — strategist: 유형 선택 및 목차 설계

```
유형 선택 기준:
- 체크리스트: ICP가 반복 수행하는 프로세스가 있을 때 (빠른 소비, 즉각 가치)
- 가이드: 복잡한 개념이나 방법론을 처음 배우는 ICP에게 (심층 교육)
- 템플릿: 즉시 복사해서 쓸 수 있는 형식이 가치 있을 때 (즉각 활용)
- 워크시트: ICP가 스스로 분석·계획해야 할 때 (참여형, 높은 몰입)

목차 설계 원칙:
1. 표지: 제목 + 부제 + 로고
2. 소개 (1페이지): 이 자료가 왜 필요한지, 무엇을 얻을 수 있는지
3. 본문 (3–7페이지): 핵심 내용
4. CTA 페이지 (1페이지): 풀 서비스·상담 유도
```

### Step 2 — copywriter: 원고 작성

```
유형별 작성 가이드:

[체크리스트]
- 항목 수: 15–30개
- 각 항목: 체크박스 + 한 줄 설명
- 카테고리별 그룹화 (3–5개 카테고리)
- 각 카테고리 앞에 짧은 설명

[가이드]
- 챕터 수: 3–5개
- 각 챕터: 소제목 + 설명(300–500자) + 실행 팁
- 시각적 요소: 박스, 표, 예시 등으로 단조로움 방지

[템플릿]
- 실제 사용 가능한 빈칸 형식
- 각 칸에 작성 가이드 포함
- 작성 예시 1개 포함 (회색 텍스트 등 구분)

[워크시트]
- 질문·분석 섹션 5–10개
- 각 섹션: 질문 + 작성 공간 + 해석 가이드
- 완성 후 다음 단계 제안

CTA 페이지 작성:
- 이 자료로 해결하지 못한 부분 1–2가지 언급
- 풀 서비스 또는 상담 신청 제안
- 할인·보너스 등 전환 인센티브 (있는 경우)
```

### Step 3 — reviewer: 검증

```
가치 제공 체크리스트:
- [ ] ICP의 핵심 고충을 실질적으로 해결하는가
- [ ] 즉시 실행 가능한 내용이 포함되어 있는가
- [ ] 전문성·신뢰성이 느껴지는가 (데이터, 사례, 출처)
- [ ] CTA가 자연스럽게 연결되는가 (강매 느낌 없이)
- [ ] 브랜드 가이드라인 준수
- [ ] 미완성 표시 없음
```

## 출력 템플릿

```markdown
---
제목: {리드마그넷 제목}
부제: {부제 — ICP 이익 중심}
유형: {체크리스트/가이드/템플릿/워크시트}
대상: {ICP 설명}
퍼널 단계: {awareness/consideration/conversion}
---

# {제목}
## {부제}

**{브랜드명} | {날짜}**

---

## 이 자료를 읽으면

- {독자가 얻는 것 1}
- {독자가 얻는 것 2}
- {독자가 얻는 것 3}

---

## {챕터/섹션 1 제목}

{내용}

---

## {챕터/섹션 2 제목}

{내용}

---

## {챕터/섹션 3 제목}

{내용}

---

## 다음 단계

{이 자료로 해결하지 못한 부분 언급}

**[{CTA 버튼 텍스트}]({CTA URL})**

{추가 인센티브 또는 보장 문구}

---
*© {연도} {회사명} | {웹사이트}*
```

## 출력 경로

```
outputs/{client}/content/{YYYYMMDD}_lead-magnet-{topic-slug}.md
```

예시:
```
outputs/flowerplus/content/20260318_lead-magnet-flower-business-checklist.md
```

**Reporter handoff:**
- 리드마그넷 성과 (다운로드 수, 전환율) → Reporter에게 `insights.md` 반영 후보 전달
- 효과적인 CTA 문구 → Reporter에게 `winning-copy.md` 반영 후보 전달
