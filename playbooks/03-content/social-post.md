# 소셜 포스트 플레이북

## 목적

LinkedIn, Instagram, X(트위터), Threads 각 플랫폼의 특성과 알고리즘에 맞는 소셜 미디어 포스트를 작성하여 브랜드 인지도를 높이고, 팔로워 인게이지먼트를 확보하며, 트래픽 및 리드를 생성한다.

> 알고리즘·수치 근거: `docs/research/2026-07-02-sns-content-research.md` (2026-07 딥리서치, 출처 포함)

**성과 KPI 기준 (2024-2026 알고리즘 반영):** 좋아요 수가 아니라 ① **도달 대비 공유 수(DM 전송)** — 비팔로워 신규 도달 최강 신호 ② **완주율/시청 시간** (릴스) ③ 저장 수 (캐러셀) — Mosseri 공식 발언 기준.

---

## 에이전트 조합

```
copywriter → reviewer
```

| 순서 | 에이전트 | 역할 | 모델 |
|------|---------|------|------|
| 1 | copywriter | 3개 플랫폼별 포스트 초안 작성 | sonnet |
| 2 | reviewer | 플랫폼 규격·브랜드 준수 여부 검증 | sonnet |

## 프레임워크

### 플랫폼별 규격

#### LinkedIn (300–500자)
```
구조: Hook → Body → CTA
- Hook (1–2줄): 스크롤을 멈추게 하는 첫 문장. 질문·통계·반직관적 주장
- Body (3–6줄): 핵심 메시지. 줄바꿈 적극 활용. 리스트 또는 번호 목록 허용
- CTA (1줄): 링크 클릭 / 댓글 유도 / 공유 요청

알고리즘 팁:
- 외부 링크는 첫 댓글에 달거나 "링크는 댓글에" 방식 사용
- 이모지는 절제하여 사용 (최대 3–5개)
- 해시태그 3–5개, 포스트 하단
```

#### Instagram (캡션 + 해시태그 3–5개)
```
구조: Hook (캡션 첫 줄) → 본문 → 해시태그 블록
- Hook (1줄, 125자 이내): "더 보기" 클릭 유도 + **검색 키워드 포함**
  (2025-07부터 공개 프로페셔널 계정 게시물이 구글에 인덱싱 — 캡션 첫 줄이 검색 자산)
- 본문 (100–300자): 이미지·영상을 보완하는 스토리 또는 정보
- 해시태그: 3–5개, 분류용 (해시태그는 도달 수단이 아님 — Mosseri 공식)
- 포맷 역할: 도달=릴스(7–30초, 첫 3초 후킹) / 저장·전환=캐러셀(1080×1350,
  1장 후킹 → 장당 1메시지 → 막장 저장 CTA) / 관계 유지=스토리

비주얼 방향:
- 이미지/영상 컨셉 1줄 설명 포함
- 캐러셀 사용 시 슬라이드별 카피 제안 (6–10장)
- 재게시·타 플랫폼 워터마크 금지 (원본 우대 정책)
```

#### X (280자 단일 또는 스레드)
```
단일 포스트 (280자):
- 핵심 메시지 하나에 집중, 첫 줄에서 주제가 즉시 드러나게
- 숫자·리스트·반전 포인트 활용
- 해시태그 1–2개만
- **링크는 본문에 넣지 않는다 — 첫 답글에 단다** (링크 포함 본문은 도달 저하,
  2024-11 머스크 공식 인정). 예약 발행 시 링크 답글까지 세트로 등록
- 좋아요보다 답글·인용을 부르는 질문형 마무리 ("여러분은 어떻게 하세요?")

스레드 (5–7포스트):
- 1번: Hook — 가장 강한 주장 또는 결론 먼저
- 2–5번: 근거·단계·사례 (각 250자 이내)
- 6번: 요약 또는 실행 팁
- 7번: CTA + 링크 (링크는 여기, 맨 끝에만)
- 각 포스트 번호 표기 (예: 1/ 2/ ...)
- 동일 문구 반복 게시 금지 (중복 감지)
```

#### Threads (1–3문장 + 이미지 1장)
```
구조: 대화 촉발형 단문
- 완성형 홍보물이 아니라 질문·짧은 의견·비하인드가 우세
- 1–3문장 + 이미지 1장 권장 (비디오 5.55% > 이미지 4.55% > 텍스트 2.79% ER — Buffer 2026)
- **태그는 1개만** (구조적으로 1개만 허용, # 미표시) — 카테고리 선언용
- 도달은 본문 키워드로 (본문 전문이 검색 대상)
- 톤: 캐주얼·구어체. 한국은 반말 소통 문화 — 브랜드 보이스로 채택 여부를
  brand-guidelines.md에서 사전 결정
- 게시 후 달리는 댓글에 반드시 답글 (답글 시 인게이지 +42% — 전 플랫폼 최대)
```
