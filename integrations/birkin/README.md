# Birkin Integration Contracts

These documents place selected Birkin skills inside AI CMO Platform workflows.
They are handoff contracts only. This repository does not automatically execute
Birkin commands, edit the Birkin repository, or claim live integration.

| Skill | AI CMO placement | Contract |
|-------|------------------|----------|
| Neurosis | Before execution when a user request is vague or high stakes. | `neurosis.md` |
| Odyssey | Before multi-step work that needs stepwise verification. | `odyssey.md` |
| Morpheus | After delivery, for maintenance notes and proposed improvements. | `morpheus.md` |
| codex-image-gen | During content or campaign workflows that need a real image. | `codex-image-gen.md` |

## Approval Rule

All consequential actions remain approval-gated:

- creating or editing client files
- running external tools
- generating or downloading images
- changing SOPs or memory
- publishing or sending buyer-facing assets
