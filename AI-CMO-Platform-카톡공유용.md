# AI CMO Platform 카톡 공유용

## 짧은 버전

AI CMO Platform은 Claude Code 기반의 AI 마케팅 운영 시스템입니다.
자연어로 지시하면 10개의 AI 에이전트가 전략, 경쟁사 분석, 블로그/SNS 작성, SEO, 세일즈 자료, 리포트까지 나눠서 처리해줍니다.

핵심은 3가지입니다.
- 33개의 core 마케팅 플레이북과 운영 chain이 들어 있어서 바로 실행됨
- 역할별 SOP와 reviewer gate가 있어서 산출물 품질을 반복 가능하게 관리함
- 클라이언트별 정보, 후속 액션, 인사이트가 누적돼서 쓸수록 정교해짐

예를 들면 이런 식으로 씁니다.
- "새 클라이언트 온보딩: 회사명, 웹사이트: URL"
- "회사명 GTM 전략 짜줘"
- "회사명 블로그 써줘"
- "회사명 경쟁사 분석해줘"
- "회사명 제안서 써줘"
- "AI CMO 운영체계로 정리해줘"

결국 마케팅팀의 반복 업무를 AI 워크플로우로 바꿔서, 기획부터 실행 문서까지 훨씬 빠르게 만드는 도구라고 보면 됩니다.

## 설명 있는 버전

AI CMO Platform은 Claude Code 안에서 돌아가는 AI 마케팅 운영체계입니다.

한 번 세팅해두면 자연어로 지시만 해도:
- GTM 전략
- 포지셔닝/경쟁사 분석
- 블로그, SNS, 뉴스레터 작성
- 콜드메일, 제안서, 피칭덱 작성
- 키워드 리서치, SEO 점검
- 성과 리포트, 회의록 정리

이런 작업을 초안 생성, 검증, 승인 기반으로 처리합니다.

구조도 단순합니다.
- 10개 전문 에이전트가 역할별로 나뉘어 일함
- 33개 SOP/플레이북이 기본 내장돼 있음
- 클라이언트별로 브랜드 톤, 카피 패턴, 가격 정책, 인사이트가 계속 쌓임
- 결과물은 폴더별로 저장되고, 다음 작업할 때 그 맥락을 다시 활용함

실사용 방식도 간단합니다.
1. Claude Code 실행
2. "새 클라이언트 온보딩: 회사명, 웹사이트: URL"
3. 이후엔 "회사명 + 원하는 작업" 형태로 요청

데이터 안전 원칙:
- GA, CRM, 매출 CSV/Excel, 고객 피드백, 미팅 메모를 넣기 전 고객명, 전화번호, 이메일, 주민/사업자번호, 계정 ID, 쿠키, 토큰, API 키, 인증 헤더를 제거하거나 마스킹합니다.
- 출력에는 원본 행, 사용자 단위 GA 데이터, private CRM row, 전체 미팅 원문을 붙이지 않고 요약/통계와 안전한 로컬 경로만 남깁니다.
- 민감정보가 제거되지 않은 자료라면 먼저 익명화 사본을 만든 뒤 작업합니다.

예:
- "플라워플러스 GTM 전략 짜줘"
- "플라워플러스 블로그 써줘. 주제는 장례식장 구독 서비스의 장점"
- "플라워플러스 콜드메일 시퀀스 만들어줘"

한마디로 정리하면, 마케팅 전략가, 콘텐츠 마케터, SEO 담당자, 세일즈 지원, 데이터 분석가 역할을 하나의 AI 운영체계로 묶어놓은 플랫폼입니다.
